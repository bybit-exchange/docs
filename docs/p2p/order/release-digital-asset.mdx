### HTTP Request
POST `/v5/p2p/order/finish`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|orderId|<b>true</b> |string |order id |


### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|[accountType](../enum#accounttype) |string |Account type|
|memberId |string |UserID|
|balance |array |Object |
|> coin |string |Currency|
|> walletBalance |string |Wallet balance|
|> transferBalance |string |Transferable balance|
|> bonus |string |Bonus|

### Request Example

```http
POST /v5/asset/transfer/query-account-coins-balance?accountType=FUND&coin=USDC HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1675866354698
X-BAPI-RECV-WINDOW: 5000
```
