### HTTP Request
POST `/v5/p2p/order/message/send`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|message|<b>true</b> |string |chat info ;text:msg;image/pdf/video: url |
|contentType|<b>true</b> |string |chat message. For text messages use: `str`; for images use: `pic`; for PDF files use: `pdf`; for video files use: `video`|
|orderId|<b>true</b> |string |order id |
|msgUuid|false |string |Client message UUID |
|fileName|false |string |image/pdf/video filename |

### Response Parameters
None

<TabItem value="python" label="Python">

```python
from bybit_p2p import P2P
import uuid

api = P2P(
    testnet=True,
    api_key="x",
    api_secret="x"
)

print(api.send_chat_message(
    message="hi, test",
    contentType="str",
    orderId="1898976123321221120",
    msgUuid=uuid.uuid4().hex
))
```
