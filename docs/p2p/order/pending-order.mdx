### HTTP Request
POST `/v5/p2p/order/pending/simplifyList`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|status|false|integer|Order status<li><code>5</code>: waiting for chain (only web3)</li><li><code>10</code>: waiting for buy pay</li><li><code>20</code>: waiting for seller release</li><li><code>30</code>: Appealing</li><li><code>90</code>: waiting buyer select tokenId</li><li><code>100</code>: objectioning</li><li><code>110</code>: Waiting for the user to raise an objection</li>|
|beginTime|false|string|begin time|
|endTime|false|string|end time|
|tokenId|false|string|token id|
|side|false|array&lt;integer&gt;|`0`: Buy, `1`: Sell|
|page|**true**|integer|Page number to query|
|size|**true**|integer|Rows to query per page|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|count |integer |Total order number |
|items |Object ||
|> id|string|order Id|
|> side|integer|Order trade type. 0: BUY, 1: SELL|
|> tokenId|string|token ID|
|> orderType|string|<li>`ORIGIN`: normal p2p order</li><li>`SMALL_COIN`: HotSwap p2p order</li><li>`WEB3`: web3 p2p order</li>|
|> amount|string|seller/buyer trade amount|
|> currencyId|string|currency ID|
|> price|string|order price|
|> fee|string|Tading fee|
|> targetNickName|string|Counterparty nickname|
|> targetUserId|string|Counterparty UID|
|> status|integer|Order status<li><code>5</code>: waiting for chain (only web3)</li><li><code>10</code>: waiting for buy pay</li><li><code>20</code>: waiting for seller release</li><li><code>30</code>: Appealing</li><li><code>90</code>: waiting buyer select tokenId</li><li><code>100</code>: objectioning</li><li><code>110</code>: Waiting for the user to raise an objection</li>|
|> createDate|string|Order created time|
|> transferLastSeconds|string|Buyer's remaining transfer time|
|> userId|string|The UID of the current query user|
|> sellerRealName|string|Seller's realname|
|> buyerRealName|string|Buyer's realname|
|> Extension|Object|Order extended information|
|>> isDelayWithdraw|boolean|`true`: delayed withdrawal|
|>> delayTime|string|Delay time|
|>> startTime|string|Delay start time|
