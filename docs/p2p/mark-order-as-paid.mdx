### HTTP Request
POST `/v5/p2p/order/pay`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|orderId|<b>true</b> |string |order id |
|paymentType|<b>true</b> |string |Payment method used |
|paymentId|<b>true</b> |string |Payment method id used |


### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |

### Request Example

```http
POST /v5/p2p/order/pay HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1675866354698
X-BAPI-RECV-WINDOW: 5000
Content-Type: application/json
{
    "orderId":"1899736339155943424",
    "paymentType":"14",
    "paymentId":"7110"
}
```
