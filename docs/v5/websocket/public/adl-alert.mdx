Query for grouped insurance pool ADL information.
> **Covers: USDT contract / USDC contract  / Inverse contract**

**Topic:**<br/>
`adlAlert.{coin}` e.g., adlAlert.USDT, adlAlert.USDC for linear, adlAlert.inverse for inverse

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list |array |Object |
|> c |string |Insurance pool currency |
|> s |string |Symbol name |
|> b |string |Insurance pool balance |
|> mb |string |Max 8h insurance pool balance |
|> pnl |string |Symbol 8h Pnl |
|> i_pr |string |ADL trigger drawdown threshold <li>When the Pnl drawdown ratio of a symbol within 8 hours exceeds this value, ADL will be triggered</li>|
|> pr |string |Symbol 8h drawdown ratio |
|> adl_tt |string |ADL trigger threshold <li>The trigger condition (an n% drawdown within 8 hours) is activated only when the insurance pool balance exceeds this threshold</li>|
|> adl_sr |string |ADL stop drawdown ratio <li>When a symbolâ€™s 8-hour drawdown ratio falls below this value, ADL will stop</li>|

### Subscribe Example

```python
{"op": "subscribe", "args": ["adlAlert.USDT"]}
```
