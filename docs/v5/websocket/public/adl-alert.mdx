Subscribe to ADL Alerts and Related Information by Grouped Insurance Pool  
> **Coverage: USDT Perpetual / USDT Delivery / USDC Perpetual / USDC Delivery / Inverse Contracts**

**Topic:**<br/>
`adlAlert.{coin}`  
For example: use `adlAlert.USDT` for USDT Perpetual/Delivery, `adlAlert.USDC` for USDC Perpetual/Delivery, and `adlAlert.inverse` for Inverse contracts.

### Response Parameters
| Parameter | Type   | Description |
|:----------|:------ |-------------|
|list       | array  | Object |
|> c        | string | Token of the insurance pool |
|> s        | string | Trading pair name |
|> b        | string | Balance of the insurance fund, used to determine if ADL is triggered |
|> mb       | string | Maximum balance of the insurance pool within the last 8h |
|> i_pr     | string | PnL ratio threshold for triggering **Contract PnL Drawdown ADL** <li>ADL is triggered when the symbol's PnL drawdown ratio within 8h exceeds this value</li> |
|> pr       | string | Symbol's PnL drawdown ratio within 8h, used to determine whether ADL is triggered or stopped |
|> adl_tt   | string | Trigger threshold for **Contract PnL Drawdown ADL** <li>This condition is only effective when the insurance pool balance is greater than this value; if so, an 8h drawdown exceeding n% may trigger ADL</li> |
|> adl_sr   | string | Stop ratio threshold for **Contract PnL Drawdown ADL** <li>ADL stops when the symbolâ€™s 8h drawdown ratio falls below this value</li> |

### Subscribe Example

```python
{"op": "subscribe", "args": ["adlAlert.USDT"]}
```
