:::info
API key needs "Earn" permission
:::

:::note
Only positions with `duration` = `Fixed` support setting auto-reinvestment. You can get the `duration` value from the response of [GET /v5/earn/product?category=OnChain](product-info.mdx).
:::

### HTTP Request
<APIEndpoint method="POST" url="/v5/earn/position/modify" />

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category|**true**|string|Product category. Fixed value: `OnChain`|
|productId|**true**|integer|Product ID. Obtained from [GET /v5/earn/product](product-info.mdx)|
|positionId|**true**|integer|Position ID. Obtained from [GET /v5/earn/position](position.mdx)|
|autoReinvest|**true**|integer|Auto-reinvestment switch. `0`: Off, `1`: On|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|retCode|integer|Return code. `0` means success|
|retMsg|string|Return message. Empty string `""` on success|
