|creationTime | int   | Data created timestamp (ms)                                                                                                              |
|data |array |Object |
| > productId | int64 | Product ID |
| > currentPrice | string | Current price |
| > buyLowPrice | array of objects | Buy low price list |
| >> selectPrice | string | Selected price |
| >> apyE8 | int64 | Annualized yield, e8 precision |
| >> maxInvestmentAmount | string | Maximum investment amount |
| > sellHighPrice | array of objects | Sell high price list |
| >> selectPrice | string | Selected price |
| >> apyE8 | int64 | Annualized yield, e8 precision |
| >> maxInvestmentAmount | string | Maximum investment amount |
