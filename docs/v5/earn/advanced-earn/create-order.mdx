:::info
Need authentication. **Up to 5 requests** per second.
:::


### HTTP Request
POST `/v5/order/advance/place-order`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category|**true**|string|Product type, e.g. `DualAssets`|
|productId|**true**|string|Product ID|
|orderType|**true**|string|Order type: `Stake`, `Redeem`|
|amount|**true**|string|Order amount (decimal string)|
|accountType|**true**|string|Account type: `FUND`, `UNIFIED`|
|coin|**true**|string|Coin name|
|orderLinkId|**true**|string|User customised order ID (max 36 characters)|
|dualAssetsExtra.orderDirection|**true**|string|Direction: `BuyLow`, `SellHigh`|
|dualAssetsExtra.selectPrice|**true**|string|Target price selected by the user|
|dualAssetsExtra.apyE8|**true**|int64|Expected APY in e8 format. If the final confirmed APY is lower than this value, the order will be rejected|
|interestCard|false|Object|Interest rate coupon|
|> awardId|**true**|int64|Interest rate coupon unique ID|
|> specCode|**true**|string|Spec_code of the interest rate coupon|


### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|orderId |string |Order ID |
|orderLinkId |string |User customised order ID |
