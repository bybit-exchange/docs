> **Covers: Margin trade (Normal Account)**

### HTTP Request
POST `/v5/spot-cross-margin-trade/repay`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|coin|**true** |string|Coin name|
|qty|false|string |Amount to repay <ul><li>`qty` is **required** when `completeRepayment`=0</li><li>`qty` is invalid when `completeRepayment`=1</li></ul>|
|completeRepayment|false |integer |Whether to pay off all debts. `0`(default): false, `1`: true|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|repayId|string|Repayment transaction ID|
