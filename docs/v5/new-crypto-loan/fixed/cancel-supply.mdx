> Permission: "Spot trade"<br/>
> UID rate limit: 1 req / second

### HTTP Request
POST `/v5/crypto-loan-fixed/supply-order-cancel`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|orderId |**true** |string |Order ID of fixed supply order|

### Response Parameters
None
