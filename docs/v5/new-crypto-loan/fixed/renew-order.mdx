> Permission: "Spot trade"<br/>
> UID rate limit: 5 req / second

### HTTP Request
GET `/v5/crypto-loan-fixed/renew-info`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|orderId|false|string|Loan order ID|
|orderCurrency|false|string|Loan coin name|
|limit |false |string |Limit for data size per page. [`1`, `100`]. Default: `10`|
|cursor |false |string |Cursor. Use the `nextPageCursor` token from the response to retrieve the next page of the result set|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list|array|Object|
|> borrowCurrency|string |Borrow currency|
|> amount|string |loan amount|
|> autoRepay|integer |`1`:Auto Repayment; `2`:Transfer to flexible loan; `0`: No Automatic Repayment. Compatible with existing orders;|
|> contractNo|string |Contract number|
|> dueTime|string |Due time|
|> orderId|integer |Order Id|
|> loanId|string |Loan Id|
|> renewLoanNo|string |Renew Loan number|
|> status |Integer |`S`: Success; `F`: Failed|
|> time|string |timestamps|
|> dto|array |Object: Loan Contract Info|
|>> apr |string |loan APR|
|>> autoRepay|integer |`1`:Auto Repayment; `2`:Transfer to flexible loan; `0`: No Automatic Repayment. Compatible with existing orders;|
|>> canLoan |Boolean |`true`: can renew loan; `false`: can not renew loan|
|>> coin|string |coin|
|>> dueTime |string |Due time|
|>> id |integer |Id|
|>> interest|string |Loan Interest|
|>> loanNo |string |Loan number|
|>> orderNo|string |Order number|
|>> orderTime |string |Order time|
|>> period |string |Period|
|>> quantity|string |Loan Quantity|
|>> remainPenaltyInterest |string |Unpaid Penalty Interest|
|>> remainPrincipal|string |Unpaid Principal|
|>> repayPrincipal |string |Principal Paid|
|>> status |string |`S`: Success; `F`: Failed|
|nextPageCursor |string |Refer to the `cursor` request parameter |
