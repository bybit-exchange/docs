Query for grouped insurance pool ADL information.
> **Covers: USDT contract / USDC contract  / Inverse contract**

### HTTP Request
GET `/v5/market/adlAlert`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------|
|symbol |false |string |Symbol name, like `BTCUSDT`, uppercase only|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|updateTime |string |Data updated timestamp (ms) |
|list |array |Object |
|> coin |string |Insurance pool currency |
|> symbol |string |Symbol name |
|> balance |string |Insurance pool balance |
|> maxBalance |string |Max 8h insurance pool balance |
|> symbolPnl |string |Symbol 8h Pnl |
|> insurancePnlRatio |string |ADL trigger drawdown threshold <li>When the Pnl drawdown ratio of a symbol within 8 hours exceeds this value, ADL will be triggered</li>|
|> pnlRatio |string |Symbol 8h drawdown ratio |
|> adlTriggerThreshold |string |ADL trigger threshold <li>The trigger condition (an n% drawdown within 8 hours) is activated only when the insurance pool balance exceeds this threshold</li>|
|> adlStopRatio |string |ADL stop drawdown ratio <li>When a symbolâ€™s 8-hour drawdown ratio falls below this value, ADL will stop</li>|
