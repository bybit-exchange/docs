Query ADL alerts and related information by grouped insurance pool  
> **Covers: USDT contract / USDC contract / Inverse contract**

:::info
- **ADL trigger and stop conditions are based on the following three scenarios:**

1. **Contract PnL Drawdown ADL**  
   - **Trigger condition**:  
     `balance` (insurance fund balance) > `adlTriggerThreshold` (trigger threshold for Contract PnL Drawdown ADL), and `pnlRatio` (8h drawdown ratio of the symbol) < `insurancePnlRatio` (PnL ratio threshold for triggering ADL).  
   - **Stop condition**:  
     `pnlRatio` (8h drawdown ratio of the symbol) > `adlStopRatio` (stop ratio threshold for ADL).  

2. **Insurance Pool Net Equity**  
   - **Trigger condition**:  
     `balance` (insurance fund balance) ≤ 0  
   - **Stop condition**:  
     `balance` (insurance fund balance) > 0  

3. **Severe Margin Loss of a `symbol` within a Grouped Pool**  
   - To ensure overall pool safety, the risk team may remove the affected symbol from the grouped insurance pool and temporarily assign it to a separate insurance pool.  
   - **Trigger condition**:  
     `balance` (insurance fund balance) ≤ 0  
   - **Stop condition**:  
     `balance` (insurance fund balance) > 0  
:::

### HTTP Request
GET `/v5/market/adlAlert`

### Request Parameters
| Parameter | Required | Type   | Description |
|:----------|:---------|:-------|-------------|
|symbol     | false    | string | Contract name, e.g. `BTCUSDT`. Uppercase only |

### Response Parameters
| Parameter | Type   | Description |
|:----------|:------ |-------------|
|updateTime | string | Latest data update timestamp (ms) |
|list       | array  | Object |
|> coin     | string | Token of the insurance pool |
|> symbol   | string | Trading pair name |
|> balance  | string | Balance of the insurance fund, used to determine if ADL is triggered |
|> maxBalance | string | Maximum balance of the insurance pool within the last 8h |
|> insurancePnlRatio | string | PnL ratio threshold for triggering **Contract PnL Drawdown ADL** <li>ADL is triggered when the symbol's PnL drawdown ratio within 8h exceeds this value</li> |
|> pnlRatio | string | `symbol`'s PnL drawdown ratio within 8h, used to determine whether ADL is triggered or stopped |
|> adlTriggerThreshold | string | Trigger threshold for **Contract PnL Drawdown ADL** <li>This condition is only effective when the insurance pool balance is greater than this value; if so, an 8h drawdown exceeding n% may trigger ADL</li> |
|> adlStopRatio | string | Stop ratio threshold for **Contract PnL Drawdown ADL** <li>ADL stops when the symbol’s 8h drawdown ratio falls below this value</li> |
