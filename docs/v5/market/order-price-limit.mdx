For spot trading order price limit, refer to <a href="https://announcements.bybit.com/en/article/title-adjustments-to-bybit-s-spot-trading-limit-order-mechanism-blt786c0c5abf865983/">announcement</a><br/>

### HTTP Request
GET `/v5/market/price-limit`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------|
|[category](../enum#category) |false |string |Product type. `spot`,`linear`,`inverse` <li>When `category` is not passed, use `linear` by default</li>|
|[symbol](../enum#symbol) |<b>true</b> |string |Symbol name, like `BTCUSDT`, uppercase only|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|symbol |string |Symbol name |
|buyLmt |string |Highest Bid Price |
|sellLmt |string |Lowest Ask Price |
|ts |string |timestamp in milliseconds |
