### HTTP Request
GET `/v5/market/time`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| timeSecond | string | Bybit server timestamp (sec)  |
| timeNano | string | Bybit server timestamp (nano)  |
