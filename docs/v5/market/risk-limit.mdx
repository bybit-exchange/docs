Query risk limit of futures
> **Covers: Linear contract / Inverse contract**

:::tip
What is risk limit? <a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=en_US&id=000001164">Risk Limit (USDT Contract)</a>
:::

### HTTP Request
GET `/v5/market/risk-limit`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |false |string |Product type. `linear`,`inverse` |
|symbol |false |string |Symbol name|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |Product type |
|list |array |Object |
|> id |integer |Risk ID |
|> symbol |string |Symbol name |
|> riskLimitValue |string |Position limit |
|> maintenanceMargin |number |Maintain margin rate |
|> initialMargin |number |Initial margin rate |
|> section |array |section array |
|> isLowestRisk |integer |`1`: true, `0`: false |
|> maxLeverage |string |Allowed max leverage |

### Request Example

```http
GET /v5/market/risk-limit?category=inverse&symbol=BTCUSD HTTP/1.1
Host: api-testnet.bybit.com
```
