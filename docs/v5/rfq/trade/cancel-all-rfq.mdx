Cancel all your inquiry. **Up to 50 requests per second ** 
:::info
<li>Inquirer cancels order:Cancel the inquiry form, all quotations are invalid</li><li> Quoter cancels the order:The inquiry form is not affected, and the quotation form is invalid</li>
:::

### HTTP Request
POST `/v5/rfq/cancel-all-rfq`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|result |object ||
|> data |Array of object |Inquiry ID |
|>> rfqId |string |Inquiry ID |
|>> rfqLinkId |string |Inquiry Custom ID |
|>> code |string |Cancel success or failure, 0 cancel success |
|>> msg |string |Cancellation failure reason |

### Request Example
