Cancel a quotation. **Up to 50 requests per second ** 
:::info
<li>Pass at least one quoteId, rfqId, and quoteLinkId.</li><li> 2. If quoteId, rfqId, and quoteLinkId are passed in at the same time, and quoteId takes precedence, the priority is judged to be quoteId > quoteLinkId > rfqId.</li>
:::

### HTTP Request
POST `/v5/rfq/cancel-quote`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|quoteId |**false** |string|Quotation ID|
|rfqId |**false** |string|Inquiry ID|
|quoteLinkId |**false**  |string |Quotation Custom ID|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|result |object ||
|rfqId |string|Inquiry ID|
|quoteId |string|Quotation ID|
|quoteLinkId|string |Quotation Custom ID|
