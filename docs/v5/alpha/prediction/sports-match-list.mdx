Query the list of sports matches available in the prediction market. Supports filtering by competition type, match status, and date range. Phase 1 supports FIFA 2026 World Cup (`eventType=1`).

### HTTP Request
<APIEndpoint method="POST" url="/v5/alpha/prediction/sports/match-list" />

### Request Parameters
| Parameter | Required | Type | Comments |
|:----- |:-------|:-----|------ |
|eventType |**true** |integer |Sports event type. Refer to [predictionEventType](../../enum#predictioneventtype-prediction-market--event-type). Phase 1: `1` (FIFA_2026)|
|status |false |integer |Match status filter. Refer to [predictionMatchStatus](../../enum#predictionmatchstatus-prediction-market--match-status)|
|startTime |false |integer |Filter matches starting after this timestamp (UTC milliseconds)|
|endTime |false |integer |Filter matches starting before this timestamp (UTC milliseconds)|
|stageCode |false |string |Filter by tournament stage. Refer to [predictionStageCode](../../enum#predictionstagecode-prediction-market--stage-code)|
|limit |false |integer |Number of records per page|
|pageIndex |false |integer |Page number starting from `1`|

### Response Parameters
| Parameter | Type | Comments |
|:----- |:-----|------ |
|list |array |Match records|
|> matchId |string |Match ID|
|> eventType |integer |Sports event type. Refer to [predictionEventType](../../enum#predictioneventtype-prediction-market--event-type)|
|> stageCode |string |Tournament stage. Refer to [predictionStageCode](../../enum#predictionstagecode-prediction-market--stage-code)|
|> homeTeam |string |Home team name|
|> awayTeam |string |Away team name|
|> matchTime |integer |Match start timestamp (UTC milliseconds)|
|> status |integer |Match status. Refer to [predictionMatchStatus](../../enum#predictionmatchstatus-prediction-market--match-status)|
|> eventId |string |Linked prediction event ID (if markets are available)|
|total |integer |Total matching records|
|pageIndex |integer |Current page number|
|pageSize |integer |Items per page|
