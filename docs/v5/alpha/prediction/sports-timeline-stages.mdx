Query the tournament timeline stages for a sports prediction event. Returns all stages with their status (Done, Active, Upcoming) and associated match counts. Useful for displaying tournament bracket progress.

### HTTP Request
<APIEndpoint method="GET" url="/v5/alpha/prediction/sports/timeline-stages" />

### Request Parameters
| Parameter | Required | Type | Comments |
|:----- |:-------|:-----|------ |
|eventType |**true** |integer |Sports event type. Refer to [predictionEventType](../../enum#predictioneventtype-prediction-market--event-type). Phase 1: `1` (FIFA_2026)|

### Response Parameters
| Parameter | Type | Comments |
|:----- |:-----|------ |
|eventType |integer |Sports event type|
|stages |array |Tournament stage timeline|
|> stageCode |string |Stage code. Refer to [predictionStageCode](../../enum#predictionstagecode-prediction-market--stage-code)|
|> stageName |string |Localized stage display name|
|> status |integer |Stage status. Refer to [predictionTimelineStatus](../../enum#predictiontimelinestatus-prediction-market--timeline-status)|
|> matchCount |integer |Total number of matches in this stage|
|> startTime |integer |Stage start timestamp (UTC milliseconds)|
|> endTime |integer |Stage end timestamp (UTC milliseconds)|
