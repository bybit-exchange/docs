To trade commidities, please sign the agreement first. Once this is done, you can trade any metals commodity contracts.

:::info
* Use master account only to sign the agreement via this endpoint, subaccount is not support to call
* Once master account is signed, all subaccounts are free to trade
* The api key with one of "Account Transfer, SubAccount Transfer, Withdrawal" permissions can call
:::

### HTTP Request
POST `/v5/user/agreement`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category | **true**|integer |`2`: Metals commodity contracts|
|agree |**true** |boolean |`true`|

### Response Parameters
None
