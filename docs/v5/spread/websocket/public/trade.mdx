Subscribe to the public trades stream.

After subscription, you will be pushed trade messages in real-time.

Push frequency: **real-time**

**Topic:**<br/>
`publicTrade.{symbol}`<br/>

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|topic |string |Topic name |
|type |string |Data type. `snapshot` |
|ts |number |The timestamp (ms) that the system generates the data |
|data |array |Object. The element in the array is sort by matching time in ascending order |
|> T |number |The timestamp (ms) that the order is filled |
|> s |string |Symbol name  |
|> S |string |Side of taker. `Buy`,`Sell` |
|> v |string |Trade size |
|> p |string |Trade price |
|> [L](../../../enum#tickdirection) |string |Direction of price change|
|> i |string |Trade ID |

### Subscribe Example

```json
{
    "op": "subscribe",
    "id": "test-001-perp",
    "args": [
        "publicTrade.SOLUSDT_SOL/USDT"
    ]
}
```

### Response Example

```json
{
    "topic": "publicTrade.SOLUSDT_SOL/USDT",
    "ts": 1744170142723,
    "type": "snapshot",
    "data": [
        {
            "T": 1744170142720,
            "s": "SOLUSDT_SOL/USDT",
            "S": "Sell",
            "v": "2.5",
            "p": "19.3928",
            "L": "MinusTick",
            "i": "31d0fc58-933b-57b3-8378-f73da06da843"
        }
    ]
}
```
