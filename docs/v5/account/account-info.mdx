Query the margin mode configuration of the account.

:::note
Query the margin mode and the upgraded status of account
:::

### HTTP Request
GET `/v5/account/info`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|[unifiedMarginStatus](../enum#unifiedmarginstatus) |integer |Account status |
|marginMode |string |`REGULAR_MARGIN`, `PORTFOLIO_MARGIN`|
|updatedTime |string |Account data updated timestamp (ms)|

### Request Example

```http
GET /v5/account/info HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1672129307221
X-BAPI-RECV-WINDOW: 5000
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "",
    "result": {
        "marginMode": "REGULAR_MARGIN",
        "updatedTime": "1672106576000",
        "unifiedMarginStatus": 3
    }
}
```
