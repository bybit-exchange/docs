Query the SMP group ID of self match prevention

### HTTP Request
GET `/v5/account/smp-group`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|smpGroup |integer |Smp group ID. If the UID has no group, it is `0` by default |
