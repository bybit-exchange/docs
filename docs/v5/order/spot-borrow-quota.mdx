Query the qty and amount of borrowable coin in spot account.
> **Covers: Spot (Unified Account)**

### HTTP Request
GET `/v5/order/spot-borrow-check`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type. `spot`|
|symbol |**true** |string |Symbol name|
|side |**true** |string |Transaction side. `Buy`,`Sell` |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|------ |
|symbol |string |Symbol name |
|side |string |Side |
|maxTradeQty |string |The maximum base coin qty can be traded<li>If spot margin trade on and symbol is margin trading pair, it returns available balance + max.borrowable amount</li><li>Otherwise, it returns actual balance</li>  |
|maxTradeAmount |string |The maximum quote coin amount can be traded<li>If spot margin trade on and symbol is margin trading pair, it returns available balance + max.borrowable amount</li><li>Otherwise, it returns actual balance</li> |
|borrowCoin |string |Borrow coin |

### Request Example

```http
GET /v5/order/spot-borrow-check?category=spot&symbol=BTCUSDT&side=Buy HTTP/1.1
Host: api-testnet.bybit.com

X-BAPI-API-KEY: XXXXX

```
