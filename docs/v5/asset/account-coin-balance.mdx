Query the balance of a specific coin in a specific [account type](../enum#accounttype). Support query sub uid's balance.

:::caution
Can query by main uid api key ONLY
:::

### HTTP Request
GET `/v5/asset/transfer/query-account-coin-balance`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|memberId |false |string |UID. **required** when query sub uid account type balance |
|[accountType](../enum#accounttype) |**true** |string |Account type |
|coin |**true** |string |Coin |
|withBonus |false |integer |`0`: not query bonus. `1`: query bonus  |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|accountType |string |Account type |
|bizType |integer |Biz type |
|accountId |string |Account ID |
|memberId |string |Uid |
|balance |Object | |
|> coin |string |Coin |
|> walletBalance |string |Wallet balance |
|> transferBalance |string |Transferable balance |
|> bonus |string |bonus |

### Request Example

```http
GET /v5/asset/transfer/query-account-coin-balance?accountType=UNIFIED&coin=BTC&memberId=592324 HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1672138188999
X-BAPI-RECV-WINDOW: 5000
```
