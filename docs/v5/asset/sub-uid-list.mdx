Query the sub UIDs under a main UID. It returns up to 2000 sub accounts, if you need more, please call this [endpoint](../user/page-subuid).

:::info
Query by the master UID's api key **only**
:::

### HTTP Request
GET `/v5/asset/transfer/query-sub-member-list`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|subMemberIds |array&lt;string&gt; |All sub UIDs under the main UID |
|transferableSubMemberIds |array&lt;string&gt; |All sub UIDs that have universal transfer enabled |
