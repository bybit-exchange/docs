Query for the list of small balance list **Up to 10 requests per second ** 

### HTTP Request
GET `/v5/asset/covert/small-balance-list`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|fromCoin |false |string |From Coin,If omitted, all will be returned by default|
|accountType |true |string |Account Type,currently only support `eb_convert_uta`|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list |array|list |
|> fromCoin |string|Currency to be converted out|
|> isConvert |integer |`1`:Support Convert `2`: Not Support|
|> availableBalance |string |Available balance of the currency to be converted|
|> usdtValue |string |The USDT value of the currency. For unsupported currencies, an empty string ("") is returned|
