Query the history of small balance  **Up to 5 requests per second ** 

### HTTP Request
GET `/v5/asset/covert/small-balance-list`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|fromCoin |false |string |From Coin,If omitted, all will be returned by default|
|accountType |true |string |UTA / funding account (enum: eb_convert_uta, eb_convert_funding)|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| list            | array | list                   |
| > accountType  | string        | Account type                                 |
| > quoteId      | string        | Quote request ID                             |
| > tradeId      | string        | Conversion transaction ID                    |
| > estAmount    | string        | Estimated total amount to be received (incl. fees) |
| >  fee          | string        | Fee                                          |
| >  feeRate      | string        | Fee rate                                     |
| >  amount       | string        | Amount credited                              |
| >  fromCoin     | string        | Sold currency                                |
| >  fromCoinAmount | string      | Amount of sold currency                      |
| >  toCoin       | string        | Received currency                            |
| >  recvAmount   | string        | Amount of received currency                  |
| nextPageCursor  | string        | Pagination cursor                            |
