Convert small balances from the coins to another coin

:::info
1. Only available (usdt, usdc, mnt)
2. ** Up to 5 requests per second ** 
:::

### HTTP Request
POST `/v5/asset/covert/small-balance-execute`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|quoteId |**true** |string |Quote request ID. Only one is supported|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| code       | int    | Result code                                                                 |
| quoteId    | string | Quote request ID                                                            |
| submitTime | string | Confirmation submission timestamp                                           |
| status     | string | Conversion status: `Success`, `Fail`, `Processing`                            |
| msg        | string | Error message when `status = Fail`                                         |
