Convert small balances from the coins to another coin

:::info
1. Only available (usdt, usdc, mnt)
2. ** Up to 5 requests per second ** 
:::

### HTTP Request
POST `/v5/asset/covert/get-quote`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|fromCoins |**true** |string |From currency. A maximum of 20 currencies can be specified in a single request, eg: ["USDT", "USDC", "XRP", "BTC"]|
|toCoin |**true** |string|For each small-amount conversion request. Only one of MNT, USDT, or USDC can be selected per request.|
|accountType |true |string |UTA / funding account (enum: eb_convert_uta, eb_convert_funding)|


### Response Parameters
| Parameter        | Type   | Comments                                                  |
|:----------------|:-------|:----------------------------------------------------------|
| quoteCreateTime | string | Quote creation timestamp (milliseconds)                   |
| ttIMS           | string | Quote expiry time                                         |
| quoteId         | string | quote request ID                                |
| fromCoinList    | string | List of currencies to be converted, e.g. USDT, USDC, XRP |
| estAmount       | string | Estimated total amount to be received (including fees)    |
| fee             | string | Fee                                                       |
| feeRate         | string | Fee rate, e.g. 2% is returned as 0.02                     |
| feeCoin         | string | Fee currency                                              |
