Set auto transfer account after deposit. The same function as the setting for Deposit on <a href="https://www.bybit.com/app/user/settings">web GUI</a>

:::info
* Your funds will be deposited into `FUND` wallet by default. You can set the wallet for auto-transfer after deposit by this API.
* Only **main** UID can access.
:::

:::tip
* [UTA2.0](../../acct-mode#uta-20) has `FUND`, `UNIFIED`
* [UTA1.0](../../acct-mode#uta-10) has `FUND`, `UNIFIED`, `CONTRACT`(for inverse derivatives)
* Classic account has `FUND`, `CONTRACT`(for inverse derivatives and derivatives), `SPOT`
:::

### HTTP Request
POST `/v5/asset/deposit/deposit-to-account`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[accountType](../../enum#accounttype)|**true** | string |Account type <ul><li><code>UNIFIED</code></li><li><code>SPOT</code></li><li><code>CONTRACT</code></li><li><code>FUND</code></li></ul>|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|status|integer |Request result: <ul><li>`1`: SUCCESS</li><li>`0`: FAIL</li></ul>|

<TabItem value="python" label="Python">

```python
from pybit.unified_trading import HTTP
session = HTTP(
    testnet=True,
    api_key="xxxxxxxxxxxxxxxxxx",
    api_secret="xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
)
print(session.set_deposit_account(
    accountType="CONTRACT",
))
```

</TabItem>

<TabItem value="Node" label="Node.js">

```n4js
const { RestClientV5 } = require('bybit-api');

const client = new RestClientV5({
  testnet: true,
  key: 'xxxxxxxxxxxxxxxxxx',
  secret: 'xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',
});

client

    accountType: 'CONTRACT'
  })
  .then((response) => {
    console.log(response);
  })
  .catch((error) => {
    console.error(error);
  });
```
