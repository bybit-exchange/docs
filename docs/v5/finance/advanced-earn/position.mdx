:::info
* Need authentication. **Up to 10 requests** per second.
* Query your active positions. Requires Earn permission on the API key.
:::

### HTTP Request
<APIEndpoint method="GET" url="/v5/earn/advance/position" />

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../../enum#advanced-earn-category) |**true**|string|Product type. `DualAssets`, `SmartLeverage`|
|productId|false|string|Product ID|
|coin |false |string |Coin. For `SmartLeverage`, filters by underlying asset|
|limit |false |int |Number of items per page. Default: 20, Max: 20|
|cursor |false |string |Pagination cursor. Use nextPageCursor from previous response|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| [category](../../enum#advanced-earn-category)|string |Product category|
| nextPageCursor | string | Cursor for the next page. Empty string if no more data |
| list|array |Object |
| > positionId | string | Position ID |
| > productId | string | Product ID. For `SmartLeverage`, returned as int64 |
| > [category](../../enum#advanced-earn-category)  | string | Product category |
| > investCoin | string | Investment coin |
| > amount | string | Position amount, unit is investCoin |
| > duration | string | Product term, e.g. `8h`, `1d`, `2d`, `3d` |
| > settlementTime | string | Settlement time, unix timestamp in ms |
| > accountType | string | Staking account: `FUND`, `UNIFIED` |
| > orderId | string | Staking order ID |
| > baseCoin | string | Base coin *(DualAssets only)* |
| > quoteCoin | string | Quote coin *(DualAssets only)* |
| > apyE8 | string | Confirmed annualized yield in e8 precision *(DualAssets only)* |
| > direction | string | `BuyLow`, `SellHigh` *(DualAssets only)*. `Long`, `Short` *(SmartLeverage only)* |
| > targetPrice | string | Target price in USDT *(DualAssets only)* |
| > [status](../../enum#advanced-earn-position-status) | string | `Active`: Staking, `Redeeming`: Redeeming *(DualAssets)*. `Active`, `Redeeming`, `Settled` *(SmartLeverage)* |
| > expectReturnCoin | string | Expected settlement coin *(DualAssets only)* |
| > expectReturnAmount | string | Expected settlement amount *(DualAssets only)* |
| > toAccountType | string | Settlement account: `FUND` *(DualAssets only)* |
| > yieldStartAt | int64 | Yield start time, unix timestamp in ms *(DualAssets only)* |
| > yieldEndAt | int64 | Yield end time, unix timestamp in ms *(DualAssets only)* |
| > underlyingAsset | string | Underlying asset, e.g. `BTC`, `ETH` *(SmartLeverage only)* |
| > leverage | string | Fixed leverage multiplier, e.g. `3`, `5` *(SmartLeverage only)* |
| > breakevenPrice | string | Breakeven price (precision-adjusted) *(SmartLeverage only)* |
| > initialPrice | string | Underlying asset price at position creation (precision-adjusted) *(SmartLeverage only)* |
| > createdTime | int64 | Position creation time, unix timestamp in ms *(SmartLeverage only)* |
| > redeemable | bool | Whether the position can be redeemed now. `false` if: global redemption is disabled, a pending redeem order exists, or less than 60 minutes remain until settlement *(SmartLeverage only)* |
| > orderLinkId | string | User customised order ID *(SmartLeverage only)* |
