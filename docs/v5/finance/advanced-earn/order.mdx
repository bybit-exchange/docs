:::info
* Need authentication. **Up to 10 requests** per second.
* Query your order history. Requires Earn permission on the API key.
:::

### HTTP Request
<APIEndpoint method="GET" url="/v5/earn/advance/order" />

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../../enum#advanced-earn-category) |**true**|string|Product type. `DualAssets`, `SmartLeverage`|
|productId|false|string|Product ID|
|orderId |false |string |Order ID|
|orderLinkId |false |string |User customised order ID|
|startTime |false |string |Start time (order creation time), unix timestamp in milliseconds|
|endTime |false |string |End time, unix timestamp in milliseconds|
|limit |false |int |Number of items per page. Default: 20, Max: 20|
|cursor |false |string |Pagination cursor. Use nextPageCursor from previous response|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| [category](../../enum#advanced-earn-category)|string |Product category|
| nextPageCursor | string | Next page cursor, empty means no more data |
| list | array | Order list |
| > orderId | string | Order ID |
| > orderLinkId | string | User customised order ID |
| > productId | string | Product ID. For `SmartLeverage`, returned as int64 |
| > [category](../../enum#advanced-earn-category)  | string | Product category |
| > orderType | string | `Stake`: Subscribe. `Redeem`: Early redemption *(SmartLeverage only)* |
| > amount | string | Order amount |
| > createdTime | string | Created time, unix timestamp in ms |
| > updatedTime | string | Updated time, unix timestamp in ms |
| > [status](../../enum#advanced-earn-order-status) | string | `Pending`: Order confirming, `Success`: Order successful (position created), `Settled`: Position settled, `Fail`: Order failed |
| > duration | string | Product term, e.g. `8h`, `1d`, `2d`, `3d` |
| > accountType | string | Account type |
| > settlementTime | int64 | Settlement time, unix timestamp in ms |
| > refundStatus | string | Refund status. Only valid when status=`Fail`: `Processing`, `Processed` |
| > coin | string | Coin name *(DualAssets only)* |
| > baseCoin | string | Base coin *(DualAssets only)* |
| > quoteCoin | string | Quote coin *(DualAssets only)* |
| > direction | string | `BuyLow`, `SellHigh` *(DualAssets only)*. `Long`, `Short` *(SmartLeverage only)* |
| > targetPrice | string | Target price in USDT *(DualAssets only)* |
| > estimateApyE8 | int64 | Estimated annualized yield, e8 precision *(DualAssets only)* |
| > toAccountType | string | Settlement destination account *(DualAssets only)* |
| > selectApyE8 | int64 | APY selected by user at order time, e8 precision *(DualAssets only)* |
| > isVip | bool | Whether it is a VIP product *(DualAssets only)* |
| > settlementCoin | string | Settlement coin. Only available when status=`Settled` *(DualAssets only)* |
| > settlementAmount | string | Settlement amount. Only available when status=`Settled` *(DualAssets only)* |
| > orderMode | string | Order mode: `Normal`, `RFQ` *(DualAssets only)* |
| > trialBonusAmount | string | Trial bonus amount *(DualAssets only)* |
| > trialBonusPnl | string | Trial bonus P&L *(DualAssets only)* |
| > investCoin | string | Investment coin, e.g. `USDT` *(SmartLeverage only)* |
| > underlyingAsset | string | Underlying asset, e.g. `BTC`, `ETH` *(SmartLeverage only)* |
| > leverage | string | Fixed leverage multiplier *(SmartLeverage only)* |
| > breakevenPrice | string | Breakeven price *(SmartLeverage only)* |
| > initialPrice | string | Underlying asset price at order time *(SmartLeverage only)* |
| > settlementPrice | string | Settlement price. Only available when status=`Settled` *(SmartLeverage only)* |
| > pnl | string | Profit and loss. Only available when status=`Settled` *(SmartLeverage only)* |
