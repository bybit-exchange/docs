:::info
* Requires Earn permission on the API key.
* The order is processed asynchronously. A successful response means the order has been accepted, not settled. Use [Get Order Info](./order) to track the order status.
* `orderLinkId` is used for idempotency — resubmitting the same `orderLinkId` returns an error indicating the order already exists.
* You must call [Get Product Quote](./product-quote) first and pass the returned quote values in `discountBuyExtra`.
* **Only `DiscountBuy` products are supported.** Ultra Discount Buy and Interest Card products are not supported by this API.
:::

### HTTP Request
<APIEndpoint method="POST" url="/v5/earn/advance/place-order" />

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category|**true**|string|Product category. `DiscountBuy`|
|productId|**true**|string|Product ID|
|orderType|**true**|string|Fixed value: `Stake`|
|amount|**true**|string|Investment amount in USDT. Precision is determined by `orderPrecisionDigital` from [Get Product Info](./product-info)|
|coin|**true**|string|Investment coin. `USDT`|
|accountType|**true**|string|Deduction account: `FUND` or `UNIFIED` (requires Unified Trading Account to be enabled)|
|orderLinkId|**true**|string|User customised order ID. Max 40 characters, alphanumeric and `_-` only|
|discountBuyExtra|**true**|Object|Discount Buy pricing parameters|
|> initialPrice|**true**|string|Current index price of the underlying asset at order time. Use `currentPrice` from [Get Product Quote](./product-quote). Max 8 decimal places|
|> purchasePrice|**true**|string|Anchor buy price from [Get Product Quote](./product-quote). Max 8 decimal places|
|> knockoutPrice|**true**|string|Knockout price from [Get Product Quote](./product-quote). Max 8 decimal places|
|> knockoutCouponE8|**true**|string|Annualized interest rate in e8 precision from [Get Product Quote](./product-quote)|
|> settleType|**true**|string|Settlement preference when exercised: `Base` = receive underlying asset (quantity = amount / purchasePrice); `Quote` = receive USDT (amount = amount / purchasePrice × settlementPrice). Has no effect upon knockout — USDT principal + interest is always returned|
|> instUid|**true**|string|Market maker institution ID from [Get Product Quote](./product-quote)|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|orderId|string|System-generated order ID|
|orderLinkId|string|User customised order ID echoed back|
