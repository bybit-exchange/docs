:::info
Does not need authentication. **Up to 50 requests** per second per IP .
:::

[Bybit Dual Asset FAQ](https://www.bybit.com/en/help-center/article/Dual-Asset-Mining-FAQ)

### HTTP Request
<APIEndpoint method="GET" url="/v5/earn/advance/product" />

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../../enum#advanced-earn-category)|**true**|string|Product category. `DualAssets`, `SmartLeverage`|
|coin|false|string|Coin name, uppercase only. For `SmartLeverage`, filters by underlying asset (e.g. `BTC`, `ETH`)|
|duration|false|string|Product duration, e.g. `8h`, `1d`, `3d`, `6d`, `12d`|
