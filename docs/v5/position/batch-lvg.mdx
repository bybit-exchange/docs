Query isolated leverage setting for futures symbols. Unlike [Get Position Info](v5/position), this endpoint does not require an open position to retrieve the leverage setting.

### HTTP Request
<APIEndpoint method="GET" url="/v5/position/symbol-info" />

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type `linear`(USDT Contract, USDC Contract), `inverse`|
|symbol |false |string |Symbol name, like `BTCUSDT`, uppercase only|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|[category](../enum#category) |string |Product type |
|list |array |Object |
|> symbol |string |Symbol name |
|> leverage |string |Leverage |
|> side |string |Meaningless field, pls ignore. `Buy`, `Sell`, `""`|
|> positionIdx |integer |Position mode. `0`: one-way; `1`: two-way Buy; `2`: two-way Sell |

:::info
Under Portfolio Margin mode, this endpoint returns an error.
:::
