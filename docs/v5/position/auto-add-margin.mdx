Turn on/off auto-add-margin for **isolated** margin position
> **Unified account covers**: USDT perpetual / USDC perpetual / USDC futures / Inverse contract<br/>
> **Classic account covers**: USDT perpetual / Inverse contract

### HTTP Request
POST `/v5/position/set-auto-add-margin`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type <ul><li>Unified account: `linear`, `inverse`</li><li>Classic account: `linear`, `inverse`</li></ul> |
|symbol |**true** |string |Symbol name |
|autoAddMargin |**true** |integer |Turn on/off. `0`: off. `1`: on |
|[positionIdx](../enum#positionidx) |false |integer |Used to identify positions in different position modes. For hedge mode position, this param is **required**<ul><li>`0`: one-way mode</li><li>`1`: hedge-mode Buy side</li><li>`2`: hedge-mode Sell side</li></ul>|

### Response Parameters
None
