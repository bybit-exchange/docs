Turn on/off auto-add-margin for **isolated** margin position

### HTTP Request
POST `/v5/position/set-auto-add-margin`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type <ul><li>[UTA2.0](../acct-mode#uta-20), [UTA1.0](../acct-mode#uta-10): `linear` (USDT Contract, USDC Contract)</li><li>Classic account: `linear` (USDT Perps)</li></ul> |
|symbol |**true** |string |Symbol name, like `BTCUSDT`, uppercase only |
|autoAddMargin |**true** |integer |Turn on/off. `0`: off. `1`: on |
|[positionIdx](../enum#positionidx) |false |integer |Used to identify positions in different position modes. For hedge mode position, this param is **required**<ul><li>`0`: one-way mode</li><li>`1`: hedge-mode Buy side</li><li>`2`: hedge-mode Sell side</li></ul>|

### Response Parameters
None
