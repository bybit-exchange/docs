Close a running futures grid trading bot. The bot will cancel all pending grid orders and close positions.

:::info
- **Bot state requirement:**<br/>
Only bots in a running state can be closed. The `bot_id` can be obtained from the [Create Futures Grid Bot](./create) response or from [Get Futures Grid Bot Detail](./get-detail).

- **After closing:**<br/>
Use [Get Futures Grid Bot Detail](./get-detail) to check the final PnL and close reason.

- **Rate limit:**<br/>
10 requests per second per UID.
:::

### HTTP Request
<APIEndpoint method="POST" url="/v5/fgridbot/close" />

### Request Parameters
| Parameter | Required | Type | Comments |
|:----------|:---------|:-----|---------|
|bot_id |**true** |string |Bot ID to close, obtained from [Create Futures Grid Bot](./create) response|

### Response Parameters
| Parameter | Type | Comments |
|:----------|:-----|---------|
|status_code |integer |`0` = success, non-zero = error|
|bot_id |string |The closed bot ID|
|debug_msg |string |Debug message (testnet only)|

---
