Query the user's card reward point balance, including available points, pending (frozen) points, account status, and related information.

### HTTP Request
<APIEndpoint method="POST" url="/v5/card/reward/points/balance" />

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|retCode |integer |Business return code. `0`: success; non-zero: failure|
|retMsg |string |Return message|
|result |object | |
|> accountId |string |Account ID|
|> availablePoint |integer |Available points|
|> pendingPoint |integer |Pending (frozen) points|
|> status |string |Account status|
|> updateTime |integer |Last update time (Unix ms timestamp)|
|> settlementPeriod |integer |Settlement period|
