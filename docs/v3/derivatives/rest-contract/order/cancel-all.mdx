:::tip
This endpoint enables to cancel all open orders.
- If `category`=_linear_, one of param `baseCoin`, `settleCoin`, `symbol` must be passed.
:::

#### HTTP Request
POST `/contract/v3/private/order/cancel-all`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |false |string |Product type. `linear`,`inverse`. Default: `linear`. Cannot cancel-all `option` order for now |
|symbol |false|string |Symbol name |
|baseCoin |false|string |Cancel all open orders by base coin. `option` ONLY |
|settleCoin |false|string |Cancel all open orders by settle coin|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|result |Object | |
|> list |array |Object |
|>> orderId |string |Order id |
|>> orderLinkId |string |User customised order id |

### Request Example

```http
POST /contract/v3/private/order/cancel-all HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1670571959925
X-BAPI-RECV-WINDOW: 5000
Content-Type: application/json
{
    "category": "linear",
    "settleCoin": "USDT",
    "symbol": null
}
```
