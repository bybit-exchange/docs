Turn on/off auto add position margin. To understand more, please read <a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=en_US&id=000001062">here</a>

:::tip
This functionality ONLY supports USDT Perpetual.
:::

### HTTP Request
POST `/contract/v3/private/position/set-auto-add-margin`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|symbol |**true** |string |Symbol name |
|side |**true** |string |Side. `Buy`,`Sell` |
|autoAddMargin |**true** |integer |Turn on/off auto add margin. `0`: off. `1`: on |
|[positionIdx](../enum#positionidx) |**true** |integer |Position index |

### Response Parameters
None

### Request Example

```http
POST /contract/v3/private/position/set-auto-add-margin HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1670826151188
X-BAPI-RECV-WINDOW: 5000
Content-Type: application/json
{
    "symbol": "ETHUSDT",
    "side": "Buy",
    "autoAddMargin": 1,
    "positionIdx": 1
}
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "OK",

    "time": 1670826151500
}
```
