Get mark price kline data

### HTTP Request
GET `/derivatives/v3/public/mark-price-kline`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |false |string |Product type. `linear`,`inverse`. Default: `linear`, but in the response category shows `""`|
|symbol |<b>true</b> |string |Symbol name|
|interval |<b>true</b> |string |Kline interval. `1` `3` `5` `15` `30` `60` `120` `240` `360` `720` `D` `M` `W`  |
|start |<b>true</b> |integer |The start timestamp (ms)|
|end |<b>true</b> |integer |The end timestamp (ms)|
|limit |false |integer |Limit for data size per page. [1,200]. Default: `200`|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|symbol |string |Symbol name |
|category |string |Product type. Keeps empty string `""` when `category` is not passed |
|list |array |<li>An string array of individual candle</li><li>Sort in reverse by `start`</li><li>The default collation within the array is `start`, `open`, `high`, `low`, `close`</li> |
