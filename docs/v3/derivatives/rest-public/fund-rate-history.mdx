### HTTP Request
GET `/derivatives/v3/public/funding/history-funding-rate`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |false |string |Product type. `linear`,`inverse`. default: `linear`, but in the response category shows `""`|
|symbol |<b>true</b> |string |Symbol name|
|start |false |integer |The start timestamp (ms)|
|end |false |integer |The end timestamp (ms)|
|limit |false |integer |Limit for data size per page. [1,200]. default: `200`|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|[category](../enum#category) |string |Product type. Keeps empty string `""` when `category` is not passed |
|list |array |Object |
|> symbol |string |Symbol name |
|> fundingRate |string |Funding rate |
|> fundingRateTimestamp |string |Funding rate timestamp(ms) |
