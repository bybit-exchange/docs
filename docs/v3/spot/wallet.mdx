### HTTP Request
GET <code>/spot/v3/private/account</code>

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| coin                 | string          | Asset        |
| coinId               | string          | Asset        |
| total                | string          | Total equity        |
| free                 | string          | Available balance |
| locked               | string          | Reserved for orders       |

### Request Example

```http
curl --location --request GET 'https://api-testnet.bybit.com/spot/v3/private/account' \
--header 'X-BAPI-SIGN: XXXXXX' \
--header 'X-BAPI-API-KEY: {api key}' \
--header 'X-BAPI-TIMESTAMP: 1659346886605' \
--header 'X-BAPI-RECV-WINDOW: 5000'
```
