#### HTTP Request
GET <code>/spot/v3/public/quote/depth/merged</code>

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|symbol | **true** | string | Name of the trading pair |
|scale | false | integer | Precision of the merged orderbook, 1 means 1 digit |
|limit | false | integer | Default value is 100. Max value is 200 |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| time | long |  Current time, unit in millisecond |
| bids | array | Bid price and quantity, with best bid prices ranked from top to bottom |
| asks | array | Ask price and quantity, with best ask prices ranked from top to bottom |

### Request Example

<Tabs>
<TabItem value="console" label="Console">

```http
curl --location --request GET 'https://api-testnet.bybit.com/spot/v3/public/quote/depth/merged?symbol=BTCUSDT&scale=1&limit=1'
```
