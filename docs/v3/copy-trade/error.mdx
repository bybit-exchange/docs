The Bybit API uses the following HTTP codes and error codes<span>:</span>


HTTP Code | t(:errorcode_meaning)
---------- | -------
200 | http_200: Request is valid -- Your request is valid
403 | http_403: Access denied -- You request too many times (refer to the <a href="#t-ipratelimits">IP rate limit</a>)
404 | http_404: Request path not found
