### HTTP Request
POST `/v5/bybitpay/refund`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|merchantId |**true** |string |Merchant UID|
|clientId |false|string |Merchant client ID|
|list|**true** | array&lt;[RefundOrderItemType](data-model/refund-order-item)&gt;|Refund order request item|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|refundOrders |array&lt;[RefundOrderType](data-model/refund-order)&gt; |Refund order details|
