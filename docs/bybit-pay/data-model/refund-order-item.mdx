|merchantTradeNo|false|string|Merchant order number, **required** if `payId` is not provided|
|payId|false|string|Bybit pay ID, **required** if `merchantTradeNo` is not provided|
|merchantRefundNo|**true**|string|Merchant refund number|
|refundAmount|**true**|string|Refund amount, support partial refund|
|env|**true**|[EnvType](env#)|Env info|
|riskInfo|false|[RiskInfoType](risk-info#)|Risk info for order pay|
|customer|false|[CustomerType](customer#)|Customer info|
