| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|refundId|false|string|Bybit refund ID, **required** if `merchantRefundNo` is not provided|
|refundType|**true**|string|Refund type <li>`MERCHNT_SELF_REFUND`: Merchant refund by Kyb fundingaccount</li><li>`STAND_SETTLEMENT_REFUND`: Settlement refund on T1, T7 report</li>|
|merchantTradeNo|**true**|string|Merchant order number|
|merchantRefundNo|false|string|Merchant refund number, **required** if `refundId` is not provided|
|payId|**true**|string|Bybit pay ID|
|refundStatus|**true**|string|Order refund status |
|amount|**true**|string|Order amount|
|createTime|**true**|number|Order creation Timestamp, in second|
