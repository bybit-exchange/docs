問卷說明

questionnaire 字段是一個 JSON 字符串，用於在提交入金發起人信息和提現接口中傳遞 Travel Rule 合規信息。對於新接入，推薦使用此字段替代舊版 beneficiary + transactionPurpose 組合。

信息
  • 最大長度：16384 字節
  • 當 questionnaire 與舊版 beneficiary/transactionPurpose 字段同時存在時，以 questionnaire 為準
  • 字段名統一使用 lowerCamelCase（如 walletType、poiIssuingCountry）
  • 國家碼遵循 ISO 3166-1 alpha-3（如 JPN、TUR、KAZ、IND）
  • 日期使用 ISO 8601 短格式 YYYY-MM-DD

公共字段​

以下字段適用於所有合規區。

字段類型是否必需說明
walletTypeinteger必填錢包類型。0：托管錢包（CASP/VASP）。1：非托管錢包（個人錢包）
legalTypestring必填實體類型。individual：自然人。company：法人/機構
vaspCodestring(64)條件必填對手方 VASP 代碼。當 walletType=0 時必填，未匹配的填 others，可通過 VASP List API 查詢
isSelfWalletbool否是否為本人自有錢包，僅非托管錢包場景使用
firstNamestring(128)條件必填自然人名（First Name）。當 legalType=individual 時必填
lastNamestring(128)條件必填自然人姓（Last Name）。當 legalType=individual 時必填
companyNamestring(256)條件必填法人/機構名稱。當 legalType=company 時必填

各合規區字段差異​

EU（歐盟，MiCA / TFR）​

公共字段 + 以下：

字段類型是否必需說明
transactionPurposestring(20,500)必填交易目的，最少 20 個字符

TUR — 土耳其（MASAK）​

公共字段 + 以下四項 POI 證件字段：

字段類型是否必需說明
poiTypestring(32)必填證件類型：ID_CARD / PASSPORT / DRIVERS / RESIDENCE_PERMIT / OTHER
poiNumberstring(64)必填證件號碼
poiIssuingCountrystring(10)必填簽發國家，三位國家碼
poiExpiredDatestring(20)必填證件有效期，YYYY-MM-DD
transactionPurposestring(20,500)必填交易目的，最少 20 個字符

KAZ — 哈薩克斯坦（AFSA）​

公共字段 + 以下四項 POI 證件字段：

字段類型是否必需說明
poiTypestring(32)必填證件類型：ID_CARD / PASSPORT / DRIVERS / RESIDENCE_PERMIT / OTHER
poiNumberstring(64)必填證件號碼
poiIssuingCountrystring(10)必填簽發國家，三位國家碼
poiExpiredDatestring(20)必填證件有效期，YYYY-MM-DD

BYBIT-IND — 印度（FIU-IND）​

僅公共字段，無額外必填項。

BYBIT-KOR — 韓國（SAFA）— 僅出金​

公共字段 + 以下：

字段類型是否必需說明
representativeFirstNamestring(128)條件必填公司代表人名（First Name）。當 legalType=company 時必填（韓國 SAFA 特殊要求）
representativeLastNamestring(128)條件必填公司代表人姓（Last Name）。當 legalType=company 時必填（韓國 SAFA 特殊要求）
備註

韓國實體入金側不採集本問卷，由本地銀行 / CTP 鏈路單獨處理。

合規區覆蓋矩陣​

實體代碼地區出金入金備注
EU歐盟✅✅MiCA / TFR
TUR土耳其✅✅MASAK，需 POI 證件
KAZ哈薩克斯坦✅✅AFSA，需 POI 證件
BYBIT-IND印度✅✅FIU-IND
BYBIT-KOR韓國✅❌SAFA，出金需公司代表人；入金走本地銀行/CTP

示例​

EU — 自然人 + 托管錢包​

{
  "walletType": 0,
  "vaspCode": "BINANCEUS_VASP",
  "legalType": "individual",
  "firstName": "John",
  "lastName": "Smith",
  "transactionPurpose": "Personal investment in long-term holdings"
}

TUR — 自然人 + 非托管錢包​

{
  "walletType": 1,
  "legalType": "individual",
  "firstName": "Ahmet",
  "lastName": "Yilmaz",
  "poiType": "ID_CARD",
  "poiNumber": "12345678901",
  "poiIssuingCountry": "TUR",
  "poiExpiredDate": "2030-06-15",
  "transactionPurpose": "Long-term holding for retirement savings purpose"
}

KAZ — 法人 + 托管錢包​

{
  "walletType": 0,
  "vaspCode": "EXAMPLE_VASP",
  "legalType": "company",
  "companyName": "Example LLC",
  "poiType": "OTHER",
  "poiNumber": "BIN1234567890",
  "poiIssuingCountry": "KAZ",
  "poiExpiredDate": "2099-12-31"
}

BYBIT-IND — 自然人 + 非托管錢包​

{
  "walletType": 1,
  "legalType": "individual",
  "firstName": "Rahul",
  "lastName": "Sharma"
}

BYBIT-KOR — 法人 + 托管錢包（僅出金）​

{
  "walletType": 0,
  "vaspCode": "EXAMPLE_VASP",
  "legalType": "company",
  "companyName": "Example Co., Ltd.",
  "representativeFirstName": "Min-jun",
  "representativeLastName": "Kim"
}
