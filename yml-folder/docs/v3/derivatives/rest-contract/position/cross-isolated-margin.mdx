Switch cross margin mode / isolated margin mode. To understand more, please read <a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=en_US&id=000001053">here</a>

:::info
Support USDT Perpetual, Inverse Perpetual and Inverse Future.
:::

### HTTP Request
POST `/contract/v3/private/position/switch-isolated`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|symbol |**true** |string |Symbol name |
|tradeMode |**true** |integer |`0`: cross margin. `1`: isolated margin |
|buyLeverage |**true** |string |Buy side leverage. Make sure `buyLeverage` equals to `sellLeverage`|
|sellLeverage |**true** |string |Sell side leverage. Make sure `buyLeverage` equals to `sellLeverage`|

### Response Parameters
None

### Request Example

```http
POST /contract/v3/private/position/switch-isolated HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX

Content-Type: application/json
{
    "symbol":"BTCUSDT",
    "tradeMode": 1,
    "buyLeverage": "8",
    "sellLeverage": "8"
}
```
