The risk limit will limit the maximum position value you can hold under different margin requirements.
If you want to hold a bigger position size, you need more margin. This interface can set the risk limit of a single
position. If the order exceeds the current risk limit when placing an order, it will be rejected. Click <a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=en_US&id=000001164">here</a> to learn more about risk limit.

:::tip
Set the risk limit of the position. You could get risk limit information of each symbol [here](../public/risk-limit#).
:::

### HTTP Request
POST `/unified/v3/private/position/set-risk-limit`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type. `linear` **only** |
|symbol |**true** |string |Symbol name|
|riskId |**true** |integer |Risk id|
|positionIdx |false |integer |Position index. `0`: one-way mode|

### Response Parameters
None

### Request Example

```http
POST /unified/v3/private/position/set-risk-limit HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1672016123862

Content-Type: application/json

{
    "category": "linear",
    "symbol": "BTCUSDT",
    "riskId": 2,
    "positionIdx": 0
}
```
