Query the configuration under the account.

### HTTP Request
GET `/unified/v3/private/account/info`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|[unifiedMarginStatus](../enum#unifiedmarginstatus) |integer |Account status |
|marginMode |string |`REGULAR_MARGIN`, `PORTFOLIO_MARGIN`|
|dcpStatus |string |[Disconnected-CancelAll-Prevention](dcp#) status: `ON`, `OFF`|
|timeWindow |integer |DCP trigger time window which user pre-set. Between [10, 300] seconds, default: 10 sec|
|smpGroup |integer |Smp group ID. If the uid has no group, it is `0` by default |
|updatedTime |string |Account data updated timestamp (ms)|

### Request Example

```http
GET /unified/v3/private/account/info HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1672023630281
X-BAPI-RECV-WINDOW: 5000
```
