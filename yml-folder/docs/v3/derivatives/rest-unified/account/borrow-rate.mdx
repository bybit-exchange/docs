### HTTP Request
GET `/unified/v3/private/account/borrow-rate`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|currency |false |string |USDC, USDT Only|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list |array |Object |
|> currency |string |Currency of all current collateral|
|> hourlyBorrowRate |string |Hourly borrow rate|
|> maxBorrowAmount |string |Max borrow amount|
|> freeBorrowAmount |string |The free interest of loan amount|

### Request Example

```http
GET /unified/v3/private/account/borrow-rate HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1673078229194
X-BAPI-RECV-WINDOW: 5000
```
