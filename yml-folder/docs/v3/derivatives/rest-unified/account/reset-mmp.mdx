:::info
Once the mmp triggered, you can unfreeze the account by this endpoint
:::

### HTTP Request
POST `/unified/v3/private/account/mmp-reset`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|baseCoin |**true** |string |Base coin|

### Response Parameters
None

### Request Example
```http
POST /unified/v3/private/account/mmp-reset HTTP/1.1
Host: api.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1675842997277
X-BAPI-RECV-WINDOW: 5000
Content-Type: application/json

{
    "baseCoin": "ETH"
}
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "success"
}
```
