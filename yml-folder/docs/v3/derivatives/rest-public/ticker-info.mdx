Get all latest information of symbols.

### HTTP Request
GET `/derivatives/v3/public/tickers`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |false |string |Product type. `linear`,`inverse`,`option`. Default: `linear`, but in the response category shows `""`|
|[symbol](../enum#symbol) |false |string |Symbol name. <ul><li>`category`=option, `symbol` is **required**</li><li>an option symbol, `category` is **required**</li></ul>|

<Tabs>
<TabItem value="console-inverse" label="inverse">

Host: api.bybit.com
```

</TabItem>

```
