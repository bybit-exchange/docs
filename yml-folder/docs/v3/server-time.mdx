Get the Bybit server timestamp

### HTTP Request
GET <code>/v3/public/time</code>

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| timeSecond | string | Bybit server timestamp (sec)  |
| timeNano | string | Bybit server timestamp (nano)  |

### Request Example

```http
curl https://api-testnet.bybit.com/v3/public/time
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "OK",
    "result": {
        "timeSecond": "1674026082",
        "timeNano": "1674026082947745743"
    },
    "retExtInfo": {},
    "time": 1674026082947
}
```
