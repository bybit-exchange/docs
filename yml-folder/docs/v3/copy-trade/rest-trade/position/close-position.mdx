Close Position

### HTTP Request
GET `/contract/v3/private/copytrading/position/close`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|symbol |**true** |string |Symbol |
|[positionIdx](../enum#positionidx) |**true** |string |Position idx, used to identify positions in different position modes: 0-Single side; 1-Buy side of both side mode; 2-Sell side of both side mode|

### Response Parameters
None

### Request Example

```http
curl --location --request POST 'https://api-testnet.bybit.com/contract/v3/private/copytrading/position/close' \
--header 'X-BAPI-SIGN-TYPE: 2' \
--header 'X-BAPI-SIGN: XXXXX' \
--header 'X-BAPI-API-KEY: XXXXX' \
--header 'X-BAPI-TIMESTAMP: 1670841189767' \
--header 'X-BAPI-RECV-WINDOW: 5000' \
--header 'Content-Type: application/json' \
--data-raw '{"symbol":"BTCUSDT","positionIdx":"1"}'
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "success",
    "result": {
    },
    "retExtInfo": {},
    "time": 1670841190031
}
```
