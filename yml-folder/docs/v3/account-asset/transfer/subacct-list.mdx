Query Subaccount List

### HTTP Request
GET `/asset/v3/private/transfer/sub-member/list/query`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|subMemberIds |array&lt;string&gt; |List of all subaccounts|
|transferableSubMemberIds|array&lt;string&gt;|List of subaccounts that are universal transfer-enabled|

### Request Example

```http
GET /asset/v3/private/transfer/sub-member/list/query HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: xxxxxxxxxxxxxx
X-BAPI-API-KEY: xxxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1670928351923
X-BAPI-RECV-WINDOW: 50000
```
