:::tip
Can query SPOT asset information only
:::

### HTTP Request
GET `/asset/v3/private/transfer/asset-info/query`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[accountType](enum#accounttype)|false |string |Account type, `SPOT` in default |
|[coin](enum#coin)|false |string |Currency type|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|spot |Object | |
|> status |string |<ul><li>`ACCOUNT_STATUS_NORMAL` -- NORMAL</li> <li>`ACCOUNT_STATUS_UNSPECIFIED` -- ABNORMAL</li></ul>|
|> assets|array|Object|
|>> [coin](enum#coin)|string |Currency type |
|>> frozen |string |Freeze amount|
|>> free |string |Free balance |
|>> withdraw |string |Amount in withdrawing |

### Request Example

```http
GET /asset/v3/private/transfer/asset-info/query?coin=USDT HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-API-KEY: XXXXXXXXXXX
X-BAPI-TIMESTAMP: 1670924059987
X-BAPI-RECV-WINDOW: 50000
X-BAPI-SIGN: XXXXXXXXXXX
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "success",
    "result": {
        "spot": {
            "status": "ACCOUNT_STATUS_NORMAL",
            "assets": [
                {
                    "coin": "USDT",
                    "frozen": "0",
                    "free": "61.1428",
                    "withdraw": ""
                }
            ]
        }
    },
    "retExtInfo": {},
    "time": 1670924060108
}
```
