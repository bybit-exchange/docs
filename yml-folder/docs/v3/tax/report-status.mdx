Get Export Report Status

### HTTP Request
POST `/fht/compliance/tax/v3/private/status`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|queryId|<b>true</b>|string|Report Request ID|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|status|string|<ul><li>`-1`: Failed</li><li>`0`: Queued</li><li>`1`: Processing</li><li>`2`: Processed</li><li>`3`: Expired</li></ul>|

### Request Example

```http
POST /fht/compliance/tax/v3/private/status HTTP/1.1
Host: api.bybit.com
X-BAPI-SIGN-TYPE: 2
X-BAPI-SIGN: xxxxxxxxxxxxxxx
X-BAPI-API-KEY: xxxxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1671183923110
X-BAPI-RECV-WINDOW: 5000
Content-Type: application/json
{"queryId":"12312312415325325"}
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "OK",
    "result": {
        "status": "2"
    },
    "retExtInfo": {},
    "time": 1671183923244
}
```
