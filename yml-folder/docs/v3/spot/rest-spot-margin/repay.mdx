### HTTP Request
POST <code>/spot/v3/private/cross-margin-repay</code>

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|coin|**true** |string|Coin name|
|qty|false|string |Amount to repay <ul><li>`qty` is **required** when `completeRepayment`=0</li><li>`qty` is invalid when `completeRepayment`=1</li></ul>|
|completeRepayment|false |integer |Whether to pay off all debts. `0`(default): false, `1`: true|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|repayId|string|Repayment transaction ID|

### Request Example

```http
POST /spot/v3/private/cross-margin-repay HTTP/1.1
Host: api-testnet.bybit.com
Content-Type: application/json
X-BAPI-API-KEY: XXXXXX
X-BAPI-TIMESTAMP: 1677751944347
X-BAPI-RECV-WINDOW: 5000
X-BAPI-SIGN: XXXXXX
Content-Length: 66

{
    "coin": "ETH",
    "qty": null,
    "completeRepayment": 1
}
```
