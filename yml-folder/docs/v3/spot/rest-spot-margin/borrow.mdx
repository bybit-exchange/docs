### HTTP Request
POST <code>/spot/v3/private/cross-margin-loan</code>

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|coin|**true**|string|Coin name|
|qty|**true**|string|Amount to borrow|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|transactId|string|Borrow transaction ID|

### Request Example

```http
POST /spot/v3/private/cross-margin-loan HTTP/1.1
Host: api-testnet.bybit.com
Content-Type: application/json
X-BAPI-API-KEY: XXXXXX
X-BAPI-TIMESTAMP: 1677751838628
X-BAPI-RECV-WINDOW: 5000
X-BAPI-SIGN: XXXXXX
Content-Length: 38

{
    "coin": "ETH",
    "qty": "10"
}
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "success",
    "result": {
        "transactId": "14143"
    },
    "retExtInfo": {},
    "time": 1662617848970
}
```
