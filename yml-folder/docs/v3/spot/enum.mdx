### timeInForce
* `GTC`&emsp;Good Till Cancel
* `FOK`&emsp;Fill or Kill
* `IOC`&emsp;Immediate or Cancel

* `BTC`

### orderStatus
* `NEW`&emsp;normal order is placed
* `ORDER_NEW`&emsp;TP/SL order is placed
* `PARTIALLY_FILLED`
* `FILLED`
* `ORDER_FILLED`&emsp;TP/SL order is filled
* `CANCELED`&emsp;normal order is cancelled
* `ORDER_CANCELED`&emsp;TP/SL order is cancelled
* `REJECTED`
* `ORDER_REJECTED`&emsp;TP/SL order is cancelled
* `PARTIALLY_FILLED_CANCELED`
* `ORDER_FAILED`&emsp;TP/SL order is failed

### ltStatus
* `1`&emsp;ETP can be purchased and redeemed
* `2`&emsp;ETP can be purchased, but not redeemed
* `3`&emsp;ETP can be redeemed, but not purchased
* `4`&emsp;ETP cannot be purchased nor redeemed

* `2`&emsp;In progress
* `3`&emsp;Failed

* `2`&emsp;Redemption
