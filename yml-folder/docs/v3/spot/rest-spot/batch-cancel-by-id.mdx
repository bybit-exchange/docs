### HTTP Request
POST <code>/spot/v3/private/cancel-orders-by-ids</code>

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
| orderIds | <b>true</b> | string | Order ID, use commas to indicate multiple orderIds. Maximum of 100 ids. |
| orderCategory | false | integer | Order category. <code>0</code>：normal order by default; <code>1</code>：<a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=us_EN&id=000001736">TP/SL order</a>, <b>Required</b> for TP/SL order.|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| list | array |Object. If all success, it returns empty array `[]` |
| > orderId | string | Order ID |
| > code | string | Errors |

### Request Example

```http
curl --location --request POST 'https://api-testnet.bybit.com/spot/v3/private/cancel-orders-by-ids' \
--header 'X-BAPI-SIGN: XXXXX' \
--header 'X-BAPI-API-KEY: {api key}' \
--header 'X-BAPI-TIMESTAMP: 1659080877551' \
--header 'X-BAPI-RECV-WINDOW: 5000' \
--header 'Content-Type: application/json' \
--data-raw '{
    "orderIds": "1210851823244553984,1210852013162639104"
}'
```
