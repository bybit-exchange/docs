> **覆蓋範圍: 全倉槓桿 (經典帳戶)**

### HTTP 請求
GET `/v5/spot-cross-margin-trade/account`

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|acctBalanceSum|string|總資產 (折算成BTC)|
|debtBalanceSum|string|總負債 (折算成BTC)|
|loanAccountList|array|Object|
|> free|string|可用餘額|
|> interest|string|未還利息|
|> loan|string|未還本金 |
|> remainAmount|string|總未還金額 = 未還利息 + 未還本金|
|> locked|string|鎖定金額 |
|> tokenId|string|幣種 |
|> total|string|合計|
|riskRate|string|風險率|
|status|integer |借貸帳戶狀態 <ul><li>`1`: 正常</li><li>`2`: 限制提幣/劃轉</li><li>`3`: 預警</li><li>`4`: 已爆倉</li></ul>|
|switchStatus|integer |全倉槓桿狀態 `0`: 關閉, `1`: 開啟|

```http
X-BAPI-API-KEY: XXXXXX
X-BAPI-TIMESTAMP: 1677751305868
X-BAPI-RECV-WINDOW: 5000
X-BAPI-SIGN: XXXXXX
```

</TabItem>

```python
    testnet=True,
    api_key="XXXXX",
```
