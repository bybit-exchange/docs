:::info
如果symbol沒有被發送，所有symbol的最佳訂單簿價格都會被返回。
:::

### HTTP 請求
GET <code>/spot/v3/public/quote/ticker/bookTicker</code>

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|symbol | **true** | string | 幣對名稱 |

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
| symbol | string | 幣對名稱 |
| bidPrice | string | 最佳買價 |
| bidQty | string | 最佳買價數量 |
| askPrice | string | 最佳賣價 |
| askQty | string | 最佳賣價數量 |
| time | number | 毫秒時間戳 |

### 請求示例

```http
curl --location --request GET 'https://api-testnet.bybit.com/spot/v3/public/quote/ticker/bookTicker?symbol=BTCUSDT'
```
