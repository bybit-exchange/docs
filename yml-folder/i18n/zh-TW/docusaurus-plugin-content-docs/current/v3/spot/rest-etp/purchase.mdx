### HTTP 請求
POST <code>/spot/v3/private/purchase</code>

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|ltCode|<b>true</b>|string|想要查詢的LT資產名稱，從資產列表中獲取.|
|ltAmount|<b>true</b>|string|下單金額|
|serialNo|false|string|序列號. 自定義ID|

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|amount|string|下單的LT資產金額|
|id|string|交易單號|
|ltCode|string|LT資產簡稱|
|orderAmount|string|實際成交金額|
|orderQuantity|string|實際成交的LT資產數量|
|[orderStatus](../enum#ltorderstatus)|string|申購單狀態：1. 已完成 2. 等待中 3. 失敗|
|serialNo|string|序列號|
|timestamp|number |時間戳|
|valueCoin|string|計價幣種|

### 請求示例

```http
curl --location --request POST 'https://api-testnet.bybit.com/spot/v3/private/purchase' \
--header 'Content-Type: application/json' \
--header 'X-BAPI-API-KEY: XXXXXXXXX' \
--header 'X-BAPI-TIMESTAMP: 1662549845240' \
--header 'X-BAPI-RECV-WINDOW: 5000' \
--header 'X-BAPI-SIGN: XXXXX' \
--data-raw '{
    "ltCode": "DOT3LUSDT",
    "ltAmount": "100",
    "serialNo": "x005"
}'
```
