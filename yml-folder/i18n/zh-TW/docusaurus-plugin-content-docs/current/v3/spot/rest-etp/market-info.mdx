### HTTP 請求
GET <code>/spot/v3/private/reference</code>

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|ltCode|<b>true</b>|string|想要查詢的LT資產名稱，從[資產信息](asset-info#)中獲取.|

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|basket |string |名義持倉（總籃子 = 籃子頭寸*發行量)|
|circulation |string |在二級市場流通的代幣數量|
|leverage |string |真實槓桿倍數|
|ltCode |string |LT資產簡稱|
|nav |string |淨值|
|navTime |number |淨值更新時間戳UTC毫秒|

### 請求示例

```http
curl --location --request GET 'https://api.bybit.com/spot/v3/private/reference?ltCode=BTC3S' \
--header 'X-BAPI-SIGN-TYPE: 2' \
--header 'X-BAPI-API-KEY: XXXXXXXXX' \
--header 'X-BAPI-TIMESTAMP: 1662549197919' \
--header 'X-BAPI-RECV-WINDOW: 5000' \
--header 'X-BAPI-SIGN: XXXXX'
```
