### HTTP 請求
POST <code>/spot/v3/private/cancel-orders</code>

### 請求參數
| 參數 | 是否必須 | 類型 | 說明 |
|:----- |:-------|:-----|------ |
| symbol | **true** | string | 幣對名稱 |
| side | false | string | 買賣方向. <code>Buy</code>, <code>Sell</code> |
| orderTypes | false | string | 訂單類型. 默認為<code>LIMIT</code>. 支持多個訂單類型取消, 用英文逗號分開, 例如：<code>LIMIT,LIMIT_MAKER</code>. |
| orderCategory | false | integer | 訂單類型. 默認為<code>0</code>：普通訂單; <code>1</code>：<a href="https://www.bybit.com/zh-TW/help-center/bybitHC_Article?language=zh_TW&id=000001736">止盈止損訂單</a>，若是止盈止損則<b>必傳</b> |

### 返回參數
| 參數 | 類型 | 說明 |
|:----- |:-----|----- |
| success | string | 是否成功. <code>0</code>：失敗, <code>1</code>：成功 |

### 請求示例

```http
curl --location --request POST 'https://api-testnet.bybit.com/spot/v3/private/cancel-orders' \
--header 'X-BAPI-SIGN: XXXXX' \
--header 'X-BAPI-API-KEY: {api key}' \
--header 'X-BAPI-TIMESTAMP: 1659078150737' \
--header 'X-BAPI-RECV-WINDOW: 5000' \
--header 'Content-Type: application/json' \
--data-raw '{
    "symbol": "BTCUSDT",
    "side": "Buy",
    "orderTypes": "LIMIT,LIMIT_MAKER"
}'
```
