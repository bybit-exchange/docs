### HTTP 請求
POST <code>/spot/v3/private/cross-margin-loan</code>

### 請求參數
| 參數 |是否必需| 類型 | 說明|
|:----- |:-------|:-----|------ |
|coin|<b>true</b>|string|幣種|
|qty|<b>true</b>|string|借幣數量|

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|transactId|string|借款交易ID|

### 請求示例

```http
POST /spot/v3/private/cross-margin-loan HTTP/1.1
Host: api-testnet.bybit.com
Content-Type: application/json
X-BAPI-API-KEY: XXXXXX
X-BAPI-TIMESTAMP: 1677751838628
X-BAPI-RECV-WINDOW: 5000
X-BAPI-SIGN: XXXXXX
Content-Length: 38

{
    "coin": "ETH",
    "qty": "10"
}
```
