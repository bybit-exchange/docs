**Topic:**<br/>
`copyTradeWallet`

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|walletBalance |number |錢包余額 |
|availableBalance |number |可用余額 = wallet balance - used margin |

### 請求訂閱

```http
ws.send('{"op": "subscribe", "args": ["copyTradeWallet"]}')
```

### 響應示例

```json
{
    "topic": "copyTradeWallet",
    "data": {
        "walletBalance": "14009.12164704",
        "availableBalance": "13998.97749271"
    }
}
```
