獲取Bybit服務器時間

### HTTP 請求
GET <code>/v3/public/time</code>

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
| timeSecond | string | Bybit服務器時間戳 (秒)  |
| timeNano | string | Bybit 服務器時間戳 (微秒)  |

### 請求示例
```http
curl https://api-testnet.bybit.com/v3/public/time
```

### 響應示例

```json
{
    "retCode": 0,
    "retMsg": "OK",
    "result": {
        "timeSecond": "1674026082",
        "timeNano": "1674026082947745743"
    },
    "retExtInfo": {},
    "time": 1674026082947
}
```
