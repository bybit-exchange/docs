## 贈金歷史

:::tip
<li><b>贈金</b> - 體驗贈金直接支付到用戶資產賬戶，可用於開倉抵扣手續費和資金費率費用。顯示的所有獎金都已被用戶應用。</li>
<li><b>優惠券</b> - 優惠券可抵扣的款項將僅用於抵消手續費。不會添加到用戶的資產賬戶中。扣除後將返還至用戶資產賬戶。 顯示的所有優惠券都已被用戶應用。</li>
:::
