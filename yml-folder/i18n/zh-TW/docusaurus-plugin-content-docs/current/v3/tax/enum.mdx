### ReportType
* `TRADE` - 交易歷史
* `P&L` - 盈虧歷史
* `EARN` - 賺取歷史
* `DEPOSIT&WITHDRAWAL` - 充提歷史
* `BONUS` - 體驗金歷史
* `AIRDROP` - 空投歷史
