### HTTP 請求
POST `/fht/compliance/tax/v3/private/status`

### 請求參數
| 參數 | 是否必需 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|queryId|<b>true</b>|string|報告請求 ID|

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|status|string |<ul><li>`-1`: 失敗</li><li>`0`: 隊列中</li><li>`1`: 處理中</li><li>`2`: 處理完成</li><li>`3`: 已過期</li></ul>|

### 請求示例

```http
POST /fht/compliance/tax/v3/private/status HTTP/1.1
Host: api.bybit.com
X-BAPI-SIGN-TYPE: 2
X-BAPI-SIGN: xxxxxxxxxxxxxxx
X-BAPI-API-KEY: xxxxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1671183923110
X-BAPI-RECV-WINDOW: 5000
Content-Type: application/json
{"queryId":"12312312415325325"}
```

### 響應示例

```json
{
    "retCode": 0,
    "retMsg": "OK",
    "result": {
        "status": "2"
    },
    "retExtInfo": {},
    "time": 1671183923244
}
```
