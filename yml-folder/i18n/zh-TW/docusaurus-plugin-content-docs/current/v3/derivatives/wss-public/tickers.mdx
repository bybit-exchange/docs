訂閱行情數據
> **覆蓋範圍: 正向合約，反向合約，期權**

:::info 重要
* 期貨，該topic推送delta數據和snapshot数据。如果delta數據為空，表明信息沒有發生變化。
* 期權只推送snapshot類型數據
:::

推送頻率: **100ms**

**Topic:**
`tickers.{symbol}`
