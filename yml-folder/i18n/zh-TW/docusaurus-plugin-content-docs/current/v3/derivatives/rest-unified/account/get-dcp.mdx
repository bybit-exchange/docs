### HTTP請求
GET `/derivatives/v3/private/query-dcp-info`

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|retCode |integer |成功/錯誤碼|
|retMsg |string |成功/錯誤信息|
|dcpStatus |string |DCP狀態. `ON`: 開啟, `OFF`: 關閉|
|timeWindow |string |設置的斷線保護時間窗口 (秒)|

### 請求示例

```http
GET /derivatives/v3/private/query-dcp-info HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1675861979020
X-BAPI-RECV-WINDOW: 50000
```

### 響應示例

```json
{
    "retCode": 0,
    "retMsg": "success",
    "dcpStatus": "ON",
    "timeWindow": "40"
}
```
