查詢某個母帳戶的子帳號列表, 返回至多2000個子帳戶, 如果您有更多, 可以調用這個[接口](../user/page-subuid).

:::caution
僅支持母帳號API key
:::

### HTTP 請求
GET `/v5/asset/transfer/query-sub-member-list`

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|subMemberIds |array&lt;string&gt; |所有子帳號 |
|transferableSubMemberIds |array&lt;string&gt; |已經開啟了萬能劃轉的子帳號 |

### 請求示例

<Tabs groupId="programming-languages">
<TabItem value="http" label="HTTP">

```http
GET /v5/asset/transfer/query-sub-member-list HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1672147239931
X-BAPI-RECV-WINDOW: 5000
```

</TabItem>
<TabItem value="python" label="Python">

```python
from pybit.unified_trading import HTTP
session = HTTP(
    testnet=True,
    api_key="XXXXX",
    api_secret="XXXXX",
)
print(session.get_sub_uid())
```

</TabItem>
</Tabs>
