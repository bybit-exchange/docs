## 1 加密（Encrypt）

### 步驟

1. 取得當前時間戳 `${timestamp}`
2. 將 `${timestamp}` 與請求體組合成新的字串 `$content`：
   - 若 HTTP 方法為 **POST**，請求體為 **JSON** 文本
   - 若 HTTP 方法為 **GET**，請求體為「**未轉義的查詢字串**」  
     - 範例：`name=foo&age=18` 為有效格式
     - `name%3dfoo%26age%3d18` 為無效格式
3. 使用 **SHA256** 與 **PKCS1V15、1024-bit RSA** 對 `$content` 進行簽名
4. 將簽名後的資料以 **Base64** 編碼

## 2 解密（Decrypt）

### 步驟

1. 從請求標頭中取得 `timestamp` 與 `signature`。
2. 取得請求體：
   - 若 HTTP 方法為 **POST**，請求體為 **JSON** 文本。
   - 若 HTTP 方法為 **GET**，請求體為「**未轉義的查詢字串**」。
3. 將 `${timestamp}` 與回應體組合成新字串 `$content`。
4. 將 `signature` 由 **Base64** 解碼為資料。
5. 使用 **SHA256** 對 `$content` 字串進行雜湊，並用 **PKCS1V15、1024-bit RSA** 驗證簽名。
