2. 請確保在報價過期之前確認報價
:::  

### HTTP 請求  
POST `/v5/fiat/trade-execute`  

### 請求參數  
| 參數               | 是否必需 | 類型   | 說明 |  
|:-------------------|:---------|:-------|------|  
| quoteTxId          | **true** | string | 報價交易 ID，來源於 [申請報價](quote-apply#response-parameters) |  
| subUserId          | **true**    | string | 用戶在 Bybit 平台的子用戶 ID |  
| webhookUrl         | false    | string | 當訂單成功或失敗時調用的 API URL（最多 256 個字符） |  
| merchantRequestId  | false    | string | 自定義請求 ID（最大長度為 36）<ul><li>通常無需填寫，但如果填寫此字段，便於內部跟踪報價請求</li></ul> |  

### 響應參數  
| 參數               | 類型     | 說明 |  
|:-------------------|:---------|------|  
| tradeNo            | string   | 交易訂單號|  
| merchantRequestId  | string   | 自定義請求 ID |
