設置充值後的自動轉入帳戶類型。該功能與<a href="https://www.bybit.com/app/user/settings">網頁端</a>-設置-充值保持一致。

:::info
* 資金會默認充值至資金帳戶, 通過該接口設置自動劃轉帳戶後，系統將會自動劃轉至目標帳戶。
* 僅支持**主帳號**調用。
:::

:::tip
* 統一帳戶2.0有`FUND`(資金錢包), `UNIFIED`(統一錢包)
* 統一帳戶1.0有`FUND`(資金錢包), `UNIFIED`(統一錢包), `CONTRACT`(反向合約錢包)
* 經典帳戶有`FUND`(資金錢包), `SPOT`(現貨錢包), `CONTRACT`(合約錢包)
:::

### HTTP 請求
POST `/v5/asset/deposit/deposit-to-account`

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|[accountType](../../enum#accounttype)|**true** | string |帳戶類型 <ul><li><code>UNIFIED</code></li><li><code>SPOT</code></li><li><code>CONTRACT</code></li><li><code>FUND</code></li></ul>|

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|status|integer |請求狀態: <ul><li>`1`: 修改成功</li><li>`0`: 修改失敗</li></ul>|

<Tabs groupId="programming-languages">
<TabItem value="http" label="HTTP">

```http

Host: api.bybit.com

X-BAPI-API-KEY: xxxxxxxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1676887913670
```

</TabItem>
<TabItem value="python" label="Python">

```python
from pybit.unified_trading import HTTP
session = HTTP(
    testnet=True,
    api_key="xxxxxxxxxxxxxxxxxx",
    api_secret="xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
)
print(session.set_deposit_account(
    accountType="CONTRACT",
))
```

</TabItem>

<TabItem value="Node" label="Node.js">

```n4js
const { RestClientV5 } = require('bybit-api');

const client = new RestClientV5({
  testnet: true,
  key: 'xxxxxxxxxxxxxxxxxx',
  secret: 'xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',
});

client
  .setDepositAccount({
    accountType: 'CONTRACT'
  })
  .then((response) => {
    console.log(response);
  })
  .catch((error) => {
    console.error(error);
  });
```
