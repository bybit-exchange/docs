### HTTP 請求
GET `/v5/asset/transfer/query-transfer-coin-list`

### HTTP 請求
| 參數 | 是否必需 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|[fromAccountType](../enum#accounttype) |**true** |string |劃出帳戶類型 |
|[toAccountType](../enum#accounttype) |**true** |string |劃入帳戶類型 |

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|list |array|幣種數組 |

### 請求示例

<Tabs>
<TabItem value="console" label="Console">

```http
GET /v5/asset/transfer/query-transfer-coin-list?fromAccountType=UNIFIED&toAccountType=CONTRACT HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1672144322595
X-BAPI-RECV-WINDOW: 5000
```

</TabItem>
</Tabs>
