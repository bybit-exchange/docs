查詢預測市場撮合引擎的可用狀態。當引擎不可用時，無法提交買入或賣出委托。

:::info
- 在提交任何買入或賣出委托前，請先調用此接口確認引擎可用
- 若 `available=false`，請勿進行交易
:::

### HTTP 請求
<APIEndpoint method="GET" url="/v5/alpha/prediction/engine-status" />

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|available |boolean |撮合引擎當前是否可用於交易|
