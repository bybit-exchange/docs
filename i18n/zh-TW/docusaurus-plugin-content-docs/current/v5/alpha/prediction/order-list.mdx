查詢已登錄用戶的預測市場委托歷史。在下達買入或賣出委托後，可調用此接口查詢最終成交狀態。FOK 委托將顯示為 `FILLED` 或 `CANCELLED`。

### HTTP 請求
<APIEndpoint method="POST" url="/v5/alpha/prediction/order-list" />

### 請求參數
| 參數 | 是否必需 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|status |false |integer |按委托狀態篩選。參考 [predictionOrderStatus](../../enum#predictionorderstatus-prediction-market--order-status)|
|tokenId |false |string |按結果代幣 ID 篩選|
|eventId |false |string |按賽事 ID 篩選|
|side |false |integer |按交易方向篩選。參考 [predictionSide](../../enum#predictionside-prediction-market--trade-direction)|
|days |false |integer |回溯天數，最大 `90`|
|limit |false |integer |每頁記錄數|
|pageIndex |false |integer |頁碼，從 `1` 開始|

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|orders |array |委托記錄|
|> orderNo |string |委托號|
|> tokenId |string |結果代幣 ID|
|> eventId |string |賽事 ID|
|> side |integer |交易方向。參考 [predictionSide](../../enum#predictionside-prediction-market--trade-direction)|
|> orderType |integer |委托類型。參考 [predictionOrderType](../../enum#predictionordertype-prediction-market--order-type)|
|> orderAmount |string |買入：投入的 USDC；賣出：賣出的份額|
|> filledSize |string |已成交份額（買入）或已賣出份額（賣出）|
|> filledPrice |string |平均成交價格|
|> totalFee |string |總手續費（USDC）|
|> status |integer |委托狀態。參考 [predictionOrderStatus](../../enum#predictionorderstatus-prediction-market--order-status)|
|> createdAt |integer |委托創建時間戳（UTC 毫秒）|
|> updatedAt |integer |最後更新時間戳（UTC 毫秒）|
|total |integer |符合條件的總記錄數|
|pageIndex |integer |當前頁碼|
