訂閱Bybit平台上最近成交的推送.<br/>
從用戶訂閱開始, 實時推送增量交易歷史, 有成交數據就推送.

推送頻率: **real-time**

**Topic:**<br/>
`publicTrade.{symbol}` **注意**: 期權使用baseCoin, e.g., publicTrade.BTC

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|topic |string |Topic名 |
|type |string |數據類型. `snapshot`,`delta` |
|ts |number |行情服務生成數據的時間戳 (毫秒) |
|data |array |Object |
|> T |number |成交時間戳 (毫秒) |
|> s |array |合約名稱  |
|> S |string |方向. `Buy`,`Sell` |
|> v |string |成交數量 |
|> p |string |成交價格 |
|> [L](../../enum#tickDirection) |string |價格變化的方向 |
|> i |string |成交Id |
|> BT |boolean |成交類型是否為大宗交易|

### 響應示例

```json
{
    "topic": "publicTrade.BTCUSDT",
    "type": "snapshot",
    "ts": 1672304486868,
    "data": [
        {
            "T": 1672304486865,
            "s": "BTCUSDT",
            "S": "Buy",
            "v": "0.001",
            "p": "16578.50",
            "L": "PlusTick",
            "i": "20f43950-d8dd-5b31-9112-a178eb6023af",
            "BT": false
        }
    ]
}
```
