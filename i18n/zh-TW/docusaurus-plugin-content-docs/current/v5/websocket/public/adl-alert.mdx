訂閱按組劃分保險池 ADL 告警及相關資訊
> **覆蓋範圍：USDT 永續 / USDT 交割 / USDC 永續 / USDC 交割 / 反向合約**

推送頻率: **1秒**

**Topic:**<br/>
`adlAlert.{coin}`

可用類型為:
- `adlAlert.USDT` 用於USDT 永續、交割
- `adlAlert.USDC` 用於USDC 永續、交割
- `adlAlert.inverse` 用於反向合約

規則詳情請參考 [查詢ADL告警](/v5/market/adl-alert) 

### 響應參數
| 參數   | 類型 | 說明 |
|:-------|:-----|----- |
|list    | array | Object |
|> c     | string | 保險池所屬幣種 |
|> s     | string | 交易對名稱 |
|> b     | string | 保險基金餘額，用於判斷是否觸發 ADL |
|> mb    | string | 最近 8 小時內的保險池最大餘額 |
|> i_pr  | string | 觸發 **合約盈虧回撤 ADL** 的盈虧比例閾值 <ul><li>當 symbol 在 8 小時內的盈虧回撤比例大於該值時，觸發 ADL</li></ul> |
|> pr    | string | symbol 在 8 小時內的回撤比例，用於判斷 ADL 是否觸發或停止 |
|> adl_tt | string | **合約盈虧回撤 ADL** 的觸發閾值 <ul><li>僅當保險池餘額大於該值時，8 小時內回撤 n% 的觸發條件才會生效</li></ul> |
|> adl_sr | string | **合約盈虧回撤 ADL** 的停止比例閾值 <ul><li>當 symbol 在 8 小時內的回撤比例小於該值時，ADL 停止</li></ul> |

### 訂閱示例

```python
{"op": "subscribe", "args": ["adlAlert.USDT"]}
```
