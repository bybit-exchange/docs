查詢按組劃分保險池ADL相關信息。
> **覆蓋範圍: USDT永續 / USDT交割 / USDC永續 / USDC交割 / 反向合約**

**Topic:**<br/>
`adlAlert.{coin}` e.g., USDT永續、交割使用adlAlert.USDT, USDC永續、交割使用adlAlert.USDC, 反向合約使用adlAlert.inverse

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|list |array |Object |
|> c |string |保險池的幣種 |
|> s |string |交易对名称 |
|> b |string |保險基金的幣種數量 |
|> mb |string |8h保险池余额最大值 |
|> pnl |string |8h内symbol的Pnl |
|> i_pr |string |觸發ADL時的回撤閾值 <li>當symbol在8h內的盈虧回撤比例大於該值時，觸發ADL</li>|
|> pr |string |symbol在8h內的回撤比例 |
|> adl_tt |string |ADL的觸發閾值 <li>只有保險池大於該值時，8h內回撤n%的觸發條件才會啟用</li>|
|> adl_sr |string |ADL的停止回撤比例 <li>當symbol在8H的回撤比例小於該值時，停止ADL</li>|

### 訂閱示例

```python
{"op": "subscribe", "args": ["adlAlert.USDT"]}
```
