訂閱Bybit平台上的強平推送

**推送頻率:** 實時

**Topic:**<br/>
`liquidation.{symbol}` e.g., liquidation.BTCUSDT

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|topic |string |Topic名 |
|type |string |數據類型. `snapshot` |
|ts |number |行情服務生成數據的時間戳 (毫秒) |
|data |Object | |
|> updatedTime |number |數據更新時間戳 (毫秒) |
|> symbol |string |合約名稱 |
|> side |string |被平的倉位方向. `Buy`,`Sell` <li>如果您收到一條Buy的推送更新, 則表面有一個多倉被強平了</li> |
|> size |string |成交數量 |
|> price |string |破產價格 |

### 訂閱示例

```python
from pybit.unified_trading import WebSocket
from time import sleep
ws = WebSocket(
    testnet=True,
    channel_type="linear",
)
def handle_message(message):
    print(message)
ws.liquidation_stream(
    symbol="BTCUSDT",
    callback=handle_message
)
while True:
    sleep(1)
```
