訂閱行情數據推送

推送頻率: **100ms**

**Topic:**<br/>
`tickers.{symbol}`

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|topic |string |Topic名 |
|type |string |數據類型. `snapshot`|
|ts |number |行情服務生成數據的時間戳 (毫秒) |
|data |map |Object |
|> symbol|string |價差產品名稱 |
|> bidPrice|string |買1價 |
|> bidSize|string |買1數量 |
|> askPrice|string |賣1價 |
|> askSize|string |賣1數量 |
|> lastPrice|string |最後成交個價 |
|> highPrice24h|string |最近24小時的最高價 |
|> lowPrice24h|string |最近24小時的最低價|
|> prevPrice24h|string |24小時前的整點市價 |
|> volume24h|string |最近24小時成交量 |

### 訂閱示例
```http
{
    "op": "subscribe",
    "args": [
        "tickers.SOLUSDT_SOL/USDT"
    ]
}
```

### 推送示例

```json
{
    "topic": "tickers.SOLUSDT_SOL/USDT",
    "ts": 1744168585009,
    "type": "snapshot",
    "data": {

        "bidPrice": "20.3359",
        "bidSize": "1.7",
        "askPrice": "",
        "askSize": "",
        "lastPrice": "21.8182",
        "highPrice24h": "24.2356",
        "lowPrice24h": "-3",
        "prevPrice24h": "22.1468",
        "volume24h": "23309.9"
    }
}
```
