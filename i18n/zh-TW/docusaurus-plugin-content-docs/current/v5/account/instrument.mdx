查詢用户全站可交易產品的基礎配置規則信息
> **覆蓋範圍:  SPOT / USDT永續 / USDT交割 / USDC永續 / USDC交割 / 反向合約**

:::caution
<!---* 現貨不支持翻頁，因此`limit`, `cusor`無效. -->
* 托管子账户不支持查询
:::

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

### HTTP請求
GET `/v5/account/instruments-info`

### 請求參數
| 參數 | 是否必需 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |<b>true</b> |string |產品類型. `spot`,`linear`,`inverse`|
|symbol |false |string |合約名稱|
|limit |false |integer |每頁數量限制. [`1`, `200`]. 默認: `200`|
|cursor |false |string |游標，用於翻頁|
