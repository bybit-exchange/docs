用戶可以自行決定是否開啟統一帳戶中幣種是否進行抵押，默認都是**開啟**的

### HTTP 請求
GET `/v5/account/set-collateral-switch`

### 請求參數
| 參數 | 類型 | 說明| 默認值 |
|:----- |:-------|:-----|------ |
|coin |**true** |string |幣種名稱 <ul><li>您可以從[這裡](collateral-info)獲取抵押品幣種</li><li>USDT, USDC不支持關閉抵押狀態</li></ul>|
|collateralSwitch |**true** |string |`ON`: 開啟抵押, `OFF`: 關閉抵押|

### 響應參數
無
