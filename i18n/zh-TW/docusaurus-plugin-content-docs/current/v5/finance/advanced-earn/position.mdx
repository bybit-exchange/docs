:::info
* 需要身份驗證。每秒**最多 10 次請求**。
* 查詢您當前生效的倉位。API 金鑰需要具備 Earn (理財) 權限。
:::

### HTTP 請求
<APIEndpoint method="GET" url="/v5/earn/advance/position" />

### 請求參數
| 參數 | 必填 | 類型 | 說明 |
|:----- |:-------|:-----|------ |
|[category](../../enum#advanced-earn-category) |**true**|string|產品類型。`DualAssets`、`SmartLeverage`|
|productId|false|string|產品 ID|
|coin |false |string |幣種。`SmartLeverage` 時用於篩選標的資產|
|limit |false |int |每頁數量。預設值：20，最大值：20|
|cursor |false |string |分頁游標。請使用上一次請求返回的 `nextPageCursor`|

### 響應參數
| 參數 | 類型 | 說明 |
|:----- |:-----|----- |
| [category](../../enum#advanced-earn-category)|string |產品類別|
| nextPageCursor | string | 下一頁的游標。如果沒有更多資料則為空字串 |
| list|array |列表 |
| > positionId | string | 倉位 ID |
| > productId | string | 產品 ID。`SmartLeverage` 時為 int64 類型 |
| > [category](../../enum#advanced-earn-category)  | string | 產品類別 |
| > investCoin | string | 投資幣種 |
| > amount | string | 倉位數量，單位為 `investCoin`（投資幣種） |
| > duration | string | 產品期限，例如 `8h`, `1d`, `2d`, `3d` |
| > settlementTime | string | 結算時間，毫秒級 Unix 時間戳 |
| > accountType | string | 申購帳戶：`FUND`（資金帳戶），`UNIFIED`（統一帳戶） |
| > orderId | string | 申購訂單 ID |
| > baseCoin | string | 基礎幣種 (Base Coin) *(僅 DualAssets)* |
| > quoteCoin | string | 計價幣種 (Quote Coin) *(僅 DualAssets)* |
| > apyE8 | string | 已確認的年化收益率（e8 精度） *(僅 DualAssets)* |
| > direction | string | 方向。`BuyLow`（低買），`SellHigh`（高賣） *(僅 DualAssets)*；`Long`（做多），`Short`（做空） *(僅 SmartLeverage)* |
| > targetPrice | string | 目標價格（掛鉤價），單位為 USDT *(僅 DualAssets)* |
| > [status](../../enum#advanced-earn-position-status) | string | `Active`（申購中），`Redeeming`（贖回中） *(DualAssets)*；`Active`，`Redeeming`，`Settled`（已結算） *(SmartLeverage)* |
| > expectReturnCoin | string | 預計結算幣種 *(僅 DualAssets)* |
| > expectReturnAmount | string | 預計結算數量 *(僅 DualAssets)* |
| > toAccountType | string | 結算帳戶：`FUND`，目前僅支援結算至資金帳戶 *(僅 DualAssets)* |
| > yieldStartAt | int64 | 計息開始時間，毫秒級 Unix 時間戳 *(僅 DualAssets)* |
| > yieldEndAt | int64 | 計息結束時間，毫秒級 Unix 時間戳 *(僅 DualAssets)* |
| > underlyingAsset | string | 標的資產，例如 `BTC`、`ETH` *(僅 SmartLeverage)* |
| > leverage | string | 固定杠桿倍數，例如 `3`、`5` *(僅 SmartLeverage)* |
| > breakevenPrice | string | 盈虧平衡價格（經精度處理） *(僅 SmartLeverage)* |
| > initialPrice | string | 建倉時標的資產價格（經精度處理） *(僅 SmartLeverage)* |
| > createdTime | int64 | 倉位創建時間，毫秒級 Unix 時間戳 *(僅 SmartLeverage)* |
| > redeemable | bool | 當前是否可贖回。以下情況返回 `false`：全局禁止贖回、存在進行中的贖回訂單、距結算時間不足 60 分鐘 *(僅 SmartLeverage)* |
| > orderLinkId | string | 用戶自定義訂單 ID *(僅 SmartLeverage)* |
