:::info
* 需要身份驗證。每秒**最多 10 次請求**。
* 查詢您的訂單歷史紀錄。API 金鑰需要具備 Earn (理財) 權限。
:::

### HTTP 請求
<APIEndpoint method="GET" url="/v5/earn/advance/order" />

### 請求參數
| 參數 | 必填 | 類型 | 說明 |
|:----- |:-------|:-----|------ |
|[category](../../enum#advanced-earn-category) |**true**|string|產品類型。`DualAssets`、`SmartLeverage`|
|productId|false|string|產品 ID|
|orderId |false |string |訂單 ID|
|orderLinkId |false |string |用戶自定義訂單 ID|
|startTime |false |string |開始時間（訂單創建時間），毫秒級 Unix 時間戳|
|endTime |false |string |結束時間，毫秒級 Unix 時間戳|
|limit |false |int |每頁數量。預設值：20，最大值：20|
|cursor |false |string |分頁游標。請使用上一次請求返回的 `nextPageCursor`|

### 響應參數
| 參數 | 類型 | 說明 |
|:----- |:-----|----- |
| [category](../../enum#advanced-earn-category)|string |產品類別|
| nextPageCursor | string | 下一頁的游標，為空字串代表沒有更多資料 |
| list | array | 訂單列表 |
| > orderId | string | 訂單 ID |
| > orderLinkId | string | 用戶自定義訂單 ID |
| > productId | string | 產品 ID。`SmartLeverage` 時為 int64 類型 |
| > [category](../../enum#advanced-earn-category)  | string | 產品類別 |
| > orderType | string | `Stake`（申購）。`Redeem`（提前贖回）僅 SmartLeverage 支援 *(SmartLeverage 支援 Redeem)* |
| > amount | string | 訂單數量 |
| > createdTime | string | 創建時間，毫秒級 Unix 時間戳 |
| > updatedTime | string | 更新時間，毫秒級 Unix 時間戳 |
| > [status](../../enum#advanced-earn-order-status) | string | `Pending`（確認中），`Success`（成功，已創建倉位），`Settled`（已結算），`Fail`（失敗） |
| > duration | string | 產品期限，例如 `8h`, `1d`, `2d`, `3d` |
| > accountType | string | 帳戶類型 |
| > settlementTime | int64 | 結算時間，毫秒級 Unix 時間戳 |
| > refundStatus | string | 退款狀態。僅在 status=`Fail` 時有效：`Processing`（處理中），`Processed`（已處理） |
| > coin | string | 幣種名稱 *(僅 DualAssets)* |
| > baseCoin | string | 基礎幣種 (Base Coin) *(僅 DualAssets)* |
| > quoteCoin | string | 計價幣種 (Quote Coin) *(僅 DualAssets)* |
| > direction | string | 方向。`BuyLow`（低買），`SellHigh`（高賣） *(僅 DualAssets)*；`Long`（做多），`Short`（做空） *(僅 SmartLeverage)* |
| > targetPrice | string | 目標價格（掛鉤價），單位為 USDT *(僅 DualAssets)* |
| > estimateApyE8 | int64 | 預估年化收益率（e8 精度） *(僅 DualAssets)* |
| > toAccountType | string | 結算目標帳戶 *(僅 DualAssets)* |
| > selectApyE8 | int64 | 用戶下單時選擇的年化收益率（e8 精度） *(僅 DualAssets)* |
| > isVip | bool | 是否為 VIP 產品 *(僅 DualAssets)* |
| > settlementCoin | string | 結算幣種，僅在 status=`Settled` 時有效 *(僅 DualAssets)* |
| > settlementAmount | string | 結算數量，僅在 status=`Settled` 時有效 *(僅 DualAssets)* |
| > orderMode | string | 訂單模式：`Normal`（普通），`RFQ`（報價請求） *(僅 DualAssets)* |
| > trialBonusAmount | string | 體驗金數量 *(僅 DualAssets)* |
| > trialBonusPnl | string | 體驗金盈虧 (P&L) *(僅 DualAssets)* |
| > investCoin | string | 投資幣種，例如 `USDT` *(僅 SmartLeverage)* |
| > underlyingAsset | string | 標的資產，例如 `BTC`、`ETH` *(僅 SmartLeverage)* |
| > leverage | string | 固定杠桿倍數 *(僅 SmartLeverage)* |
| > breakevenPrice | string | 盈虧平衡價格 *(僅 SmartLeverage)* |
| > initialPrice | string | 下單時標的資產價格 *(僅 SmartLeverage)* |
| > settlementPrice | string | 結算價格，僅在 status=`Settled` 時有值 *(僅 SmartLeverage)* |
| > pnl | string | 收益（盈虧），僅在 status=`Settled` 時有值 *(僅 SmartLeverage)* |
