## 交易所經紀商API頻率規則
預設情況下，交易所經紀商（Exchange Broker）實體底下的所有帳戶都適用於標準的 API 速率限制。
若您基於業務需求有特殊要求，請聯繫您的**客戶經理**（Account Manager） 申請更高的 Exchange Broker 層級上限以及子帳戶 UID 的速率限制上限。審核通過後，您即可依需求將速率限制配額分配至各個子帳戶。

:::info
- 所有現有子帳戶仍享有原有的 API 速率限制。
- 新子帳戶的預設 API 速率限制不計入EB API 速率限制總配額。
- 如果超出EB API 速率限制總配額，您只能先降低帳戶的 API 速率限制。當 API 速率限制總配額低於機構 API 速率限制總配額後，您才能提高帳戶的 API 速率限制。
:::
