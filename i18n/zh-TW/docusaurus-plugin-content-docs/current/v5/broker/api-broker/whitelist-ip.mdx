:::info
此接口必須從白名單 IP 位址發起請求。
:::

### HTTP 請求
<APIEndpoint method="GET" url="/v5/broker/whitelist/ip" />

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明 |
|:----- |:-----|------ |
|result |array |Object |
|> name |string |客戶名稱 |
|> endpointDomain |string |分配給該客戶專用的 hostname |
|> ipList |array |白名單 IP 位址列表 |

### 請求示例

<Tabs groupId="programming-languages">
<TabItem value="http" label="HTTP">

```http
GET /v5/broker/whitelist/ip HTTP/1.1
Host: api.bybit.com
```

</TabItem>
</Tabs>
