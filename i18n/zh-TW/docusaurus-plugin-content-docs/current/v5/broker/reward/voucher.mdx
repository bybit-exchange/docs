### HTTP 請求
POST `/v5/broker/award/info`

### 請求參數
| 參數 | 是否必需 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|id |**true** |string |代金券ID|

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|id |string |代金券ID|
|coin |string |幣種|
|amountUnit |string |計價單位<ul><li><code>AWARD_AMOUNT_UNIT_USD</code></li><li><code>AWARD_AMOUNT_UNIT_COIN</code></li></ul>|
|productLine |string |業務線|
|subProductLine |string |子業務線|
|totalAmount |Object |代金券總金額|
|usedAmount |string |代金券已發出金額|

<TabItem value="python" label="Python">

```python

session = HTTP(
    testnet=True,
    api_key="xxxxxxxxxxxxxxxxxx",
    api_secret="xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
)
print(session.get_voucher_spec(
    id="80209",
))
```

</TabItem>

<TabItem value="Node" label="Node.js">

```n4js
const { RestClientV5 } = require('bybit-api');

const client = new RestClientV5({
  testnet: true,
  key: 'xxxxxxxxxxxxxxxxxx',
  secret: 'xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',
});

client
  .getBrokerVoucherSpec({
    accountId: '5714139',
    awardId: '189528',
    specCode: 'demo000',
    withUsedAmount: false,
})
  .then((response) => {
    console.log(response);
  })
  .catch((error) => {
    console.error(error);
  });
```
