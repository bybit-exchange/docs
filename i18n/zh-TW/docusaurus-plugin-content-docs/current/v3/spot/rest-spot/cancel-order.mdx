### HTTP 請求
POST <code>/spot/v3/private/cancel-order</code>

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
| orderId | false | string | 訂單ID. 當<code>orderLinkId</code>為空時, 則<b>必傳</b> |
| orderLinkId | false | string | 用戶自定義ID. 當<code>orderId</code>為空時, 則<b>必傳</b> |
| orderCategory | false | integer | 訂單類型. 默認為<code>0</code>：普通訂單; <code>1</code>：<a href="https://www.bybit.com/zh-TW/help-center/bybitHC_Article?language=zh_TW&id=000001736">止盈止損訂單</a>，若是止盈止損則<b>必傳</b> |

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
| orderId | string| 訂單ID |
| orderLinkId | string| 自定義訂單ID，用戶自己生成 |
| symbol | string| 幣對名稱 |
| [status](../enum#orderstatus) | string| 訂單狀態 |
| accountId | string| 賬戶ID |
| orderPrice | string| 訂單價格 |
| createTime | string| 訂單創建時間 |
| orderQty | string| 訂單數量 |
| execQty | string| 成交數量 |
| [timeInForce](../enumtimeinforce) | string| <a href="https://www.bybit.com/zh-TW/help-center/bybitHC_Article?language=zh_TW&id=000001044">執行策略</a> |
| orderType | string| 訂單類型, LIMIT/MARKET/LIMIT_MAKER |
| [side](../enuM#side) | string| 買賣方向. <code>BUY</code>：買入, <code>SELL</code>：賣出. |

### 請求示例

```http
curl --location --request POST 'https://api-testnet.bybit.com/spot/v3/private/cancel-order' \
--header 'X-BAPI-SIGN: XXXXX' \
--header 'X-BAPI-API-KEY: {api key}' \
--header 'X-BAPI-TIMESTAMP: 1659078017415' \
--header 'X-BAPI-RECV-WINDOW: 5000' \
--header 'Content-Type: application/json' \
--data-raw '{
    "orderId": "1210810256551063296",
    "orderLinkId": null
}'
```
