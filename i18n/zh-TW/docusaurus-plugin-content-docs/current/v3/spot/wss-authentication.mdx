<p>構建連接時，創建鑒權請求。</p>
<p>注意: 如果您正在使用<a href="https://github.com/bybit-exchange/pybit">pybit</a>, <a href="https://www.npmjs.com/package/bybit-api">bybit-api</a>或者其他第三方庫, 您可以忽略此項-因為鑒權已經內建。</p>

```json
{
    "req_id": "10001", // 可選
    "op": "auth",
    "args": [
        "api_key",
        1662350400000, //過期時間應當大於當前時間戳
        "singature"
    ]
}
```

```python
# 基於: https://github.com/bybit-exchange/pybit/blob/master/pybit/_http_manager.py

import hmac
import json
import time
import websocket

api_key = ""
api_secret = ""

# 生成expires
expires = int((time.time() + 1) * 1000)

# 生成簽名
signature = str(hmac.new(
    bytes(api_secret, "utf-8"),
    bytes(f"GET/realtime{expires}", "utf-8"), digestmod="sha256"
).hexdigest())

ws = websocket.WebSocketApp(
    url=url,
    ...
)

# 發送鑒權請求
ws.send(
    json.dumps({
        "op": "auth",
        "args": [api_key, expires, signature]
    })
)
```

> 鑒權成功的響應示例

```json
{
    "success": true,
    "ret_msg": "",
    "conn_id": "06a3cbfffe5df9cb-0000000e-00026965-fe306ee9ec20f544-4ab7116c",
    "req_id": "",
    "op": "auth"
}
```

:::info
簽名生成的示例可以參考<a href="https://github.com/bybit-exchange/api-usage-examples">這裡</a>。
:::

:::caution
由於網絡的複雜性，您可能隨時遇到斷連。請參考以下建議確保您能即時接收到推送：
1. 通過發送[心跳](connect#如何發送心跳)
2. 遇到斷連時，立即重新連接
:::


## 如何發送心跳

> 發送ping的示例

```javascript
// req_id是可選參數
ws.send(JSON.stringify({"req_id": "100001", "op": "ping"}));
```

> 接收到pong的響應示例

```json
// 有req_id
{
    "op": "pong",
    "args": [
        1661741630642
    ],
    "req_id": "100001",
    "conn_id": "706b870c"
}

// 沒有req_id
{
    "op": "pong",
    "args": [
        1661741654529
    ],
    "conn_id": "706b870c"
}
```

:::caution
為了維持連接，我們推薦您每**20秒**發送一次心跳。
:::

## 頻率限制
**僅限私有頻道：**
* 每個api_key最多可以保持100個websocket私有連接，超過100個連接後的新連接會被服務器拒絕。
* 單個IP同時最多50個websocket私有topic鏈接

## 如何訂閱topic
### 理解Websocket裡的args

> 通過傳入args來訂閱指定topic

```json
// 訂閱trade下的BTCUSDT
{
    "req_id": "10001", // 可選
    "op": "subscribe",
    "args": [
        "trade.BTCUSDT"
    ]
}
```

> 通過","來訂閱同一個topic下的多個交易對

```json
{
    "req_id": "10001", // 可選
    "op": "subscribe",
    "args": [
        "trade.BTCUSDT",
        "trade.ETHUSDT"
    ]
}
```

> 訂閱多個topic

```javascript
ws.send('{"req_id": "10001", "op": "subscribe", "args": ["trade.BTCUSDT", "trade.ETHUSDT", "tickers.BTCUSDT", "bookticker.BTCUSDT"]}')
```

:::caution
您最多傳遞10個參數到args裡，超過則報錯
:::

### ### 理解如何取消訂閱

> 您可以通過發送請求來動態地停止訂閱

```javascript
ws.send('{"req_id": "10002", "op": "unsubscribe", "args": ["trade.BTCUSDT"]}');
```

```json
// 示例: 取消訂閱BTCUSDT和ETHUSDT
{
    "req_id": "10002", // 可選
    "op": "unsubscribe",
    "args": [
        "trade.BTCUSDT",
        "trade.ETHUSDT"
    ]
}
```

```json
// 取消訂閱的響應
{
    "op": "unsubscribe",
    "success": true,
    "req_id": "10002",
    "ret_msg": "unsubscribe",
    "conn_id": "46f097b7"
}
```
