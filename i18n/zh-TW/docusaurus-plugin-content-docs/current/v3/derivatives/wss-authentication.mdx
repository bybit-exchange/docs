## 鑒權/認證
:::info
**公共頻道**不需要鑒權，以下部分僅適用於**私有頻道**的訂閱。
:::

<p>構建連接時，創建鑒權請求。</p>
<p>注意: 如果您正在使用<a href="https://github.com/bybit-exchange/pybit">pybit</a>, <a href="https://www.npmjs.com/package/bybit-api">bybit-api</a>或者其他第三方庫, 您可以忽略此項-因為鑒權已經內建。</p>

```json
{
    "req_id": "10001", // 可選項
    "op": "auth",
    "args": [
        "api_key",
        1662350400000, //過期時間應當大於當前時間戳
        "singature"
    ]
}
```

```py
# 基於: https://github.com/bybit-exchange/pybit/blob/master/pybit/_http_manager.py

import hmac
import json
import time
import websocket

api_key = ""
api_secret = ""

# Generate expires.
expires = int((time.time() + 1) * 1000)

# Generate signature.
signature = str(hmac.new(
    bytes(api_secret, "utf-8"),
    bytes(f"GET/realtime{expires}", "utf-8"), digestmod="sha256"
).hexdigest())

ws = websocket.WebSocketApp(
    url=url,
    ...
)

# Authenticate with API.
ws.send(
    json.dumps({
        "op": "auth",
        "args": [api_key, expires, signature]
    })
)
```

## 如何訂閱topic
### 理解Websocket裡的args

> 通過傳入args來訂閱指定topic

```json
// 訂閱1檔深度推送
{
    "req_id": "test", // 可選
    "op": "subscribe",
    "args": [
        "orderbook.1.BTCUSDT"
    ]
}
```

> 通過逗號隔開，可以同時訂閱多個topic或者多個symbol

```json
{
    "req_id": "test", // 可選
    "op": "subscribe",
    "args": [
        "orderbook.1.BTCUSDT",
        "publicTrade.BTCUSDT"，
        "orderbook.1.ETHUSDT",
    ]
}
```

### 理解如何取消訂閱

> 您可以通過發送請求來動態地停止訂閱

```json
{
    "op": "unsubscribe",
    "args": [
        "publicTrade.ETHUSD"
    ],
    "req_id": "customised_id"
}
```
